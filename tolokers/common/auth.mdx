# Registration and logging in

## I am registered in Toloka {#registered}

1. Open [Toloka](http://toloka.yandex.com/).
1. Choose an account and tap **Continue** or enter your username and password.

If you don't remember your username or password, go to the [Restore access](https://passport.yandex.com/passport?mode=restore) page.

{% if platform == "ios" %}
Once you log in to the app, begin [completing tasks](/docs-new/tolokers/tasks).
{% elsif platform == "android" %}
Once you log in to the app, [adjust the settings](/docs-new/tolokers/settings) and begin [completing tasks](/docs-new/tolokers/tasks).
{% endif %}

## I am registered in Yandex, but not in Toloka {#no-toloka}

1. Open Toloka.
1. Choose an account or enter your username and password.
1. If a warning appears on the registration screen:

   <Cut title="No phone number linked">

   {% if platform == "ios" %}<Image withBackground imageId="1676448733221" alt="" />{% elsif platform == "android" %}<Image withBackground imageId="1676448934029" alt="" />{% endif %}

   Tap {% if platform == "ios" %}**Go**{% elsif platform == "android" %}**OK**{% endif %} and link the phone number on the page that opens in [Yandex ID](https://passport.yandex.com/). We need your number to send the SMS code.

   </Cut>

   <Cut title="The phone number is linked to another account">

   {% if platform == "ios" %}<Image withBackground imageId="1676448725030" alt="" />{% elsif platform == "android" %}<Image withBackground imageId="1676448860301" alt="" />{% endif %}

   A user with the phone number specified in your account is already registered in Toloka. Check the number in [Yandex ID](https://passport.yandex.com/profile/phones).

   If the wrong number is linked, change it. For more information about linking a phone number, see [Yandex ID Help](https://yandex.com/support/passport/authorization/phone.html).

   If the number is correct, it means you haven't registered in Toloka before with another username. Use [Restoring access](https://passport.yandex.com/passport?mode=restore).

   <Note mode="warning" title="Warning">

   According to the [User Agreement](https://toloka.ai/tolokers/legal/user-agreement/), a user can have only one account in Toloka.

   </Note>

   </Cut>

1. Fill in the profile information.

1. Accept the User Agreement and click {% if platform == "ios" %}**Done**{% elsif platform == "android" %}**Start earning money**{% endif %}.

{% if platform == "ios" %}Once you log in to the app, begin [completing tasks](/docs-new/tolokers/tasks).{% endif %}

{% if platform == "android" %}Once you log in to the app, [adjust the settings](/docs-new/tolokers/settings) and begin [completing tasks](/docs-new/tolokers/tasks).{% endif %}

## I don't have a Yandex ID {#no-yandex}

1. Open Toloka.
1. Click **Create ID**.
1. Register in Yandex. Be sure to enter your mobile phone number. We need it for sending the SMS code.
1. [Register](#no-toloka) in Toloka.
