# Messages

In [Messages](http://toloka.yandex.com/messages) you receive:

- Notifications about transactions on your account.
- News and notifications about the platform.
- Messages from the requester about training completion, skill assignment, new tasks and rewards.

Write to the requester:

- If a task doesn't work correctly (it shows a blank screen, doesn't display images, the <If condition={props.platform == "web"}>**Submit**</If><If condition={props.platform == "ios"}>**Submit**</If><If condition={props.platform == "android"}>**Done**</If> button doesn't work, and so on).

- To ask about the instructions, training or the skill level.
