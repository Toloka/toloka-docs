<Cut title="How can I speed up the pool completion?">

- To motivate Tolokers, assign a [public skill](https://toloka.ai/en/docs/guide/concepts/nav-create#public) and use [dynamic pricing](https://toloka.ai/en/docs/guide/concepts/dynamic-pricing).

- Try to [increase the project rating](https://toloka.ai/en/docs/guide/concepts/project_rating_stat), so that your task is higher in the list of tasks for Tolokers.

- Adjust the [quality-speed ratio](https://toloka.ai/en/docs/guide/concepts/adjust).

- Set a higher [priority](https://toloka.ai/en/docs/guide/concepts/pool_poolparams#priority) for the pool among other project pools.
