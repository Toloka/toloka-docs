<Cut title="My Toloka token expired. How or where can I find its lifetime?">

You can see the token expiration time on the **Integrations** tab of the [Profile](https://platform.toloka.ai/requester/profile).

Click **Get the OAuth-token** to get a new token.

If token expiration time is `0`:

1. Go to Yandex ID.
2. Log out from your account.
3. Log in Toloka again.

</Cut>
