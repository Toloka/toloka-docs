In the Toloka ML platform interface, a new version of a dataset is added created time you label the dataset.

1. Click the dataset table row in the [Datasets](https://tolokamodels.tech/datasets) section table.

1. Click **Annotate this version** in the upper-right area. The **Data labeling** page opens.

1. Select dataset type from the list of available ones (image, text, or two texts). Check the **Multilabel** option to allow more than one output label.

1. Select the column names present in your dataset file both for data and label and click **Next step**.

1. Check the correctness of the autoloaded labels, edit them if needed or add new ones, and click **Next step**.

1. Check if your data is labeled correctly. Click the label name if you need to change it.

1. Click **Next page >** in case you have multiple pages.

1. When finished, click **Complete**. A new version of dataset will be saved and accessible in the **Version:** list on the dataset page.
