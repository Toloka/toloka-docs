# ModelsClient

## Model fields

* `uid`: Unique ID.
* `parent_uid`: [Parent model](/docs/toloka-ml/models/collections#foundational-models) used as the basis for creating the current model.
* `run_id`: ID of the run the model is created in.
* `name`: Model name.
* `version`: Version.
* `description`: Description.
* `tags`: List of tags.
* `status`: Model status.
* `model_type`: Model type.
* `model_meta`: Model metadata with arbitrary fields.
* `artifacts`: List of links to model artifacts.
