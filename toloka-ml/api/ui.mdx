# PulsarUIClient

A class for working with the Toloka ML platform UI.

## Parameters {#parameters}

* `token=None`: `string`: (Optional) [OAuth token](/docs/toloka-ml/api/#token). By default, the token is taken from the `~/.pulsar/token` file.

## Methods {#methods}

<Cut title="create_object_table_config(self, sorting, filtering, column_bands, column_width, filter_logic, row_height, sync_aggregation)">

Creating a table with object parameters.

Parameters:

* `sorting=None`: `Sorting` or `list[Sorting]`: (Optional) sorting rules.
* `filtering=None`: `Filtering` or `list[Filtering]`: (Optional) filtering rules.
* `column_bands=None`: `list[string]`: (Optional) columns to visualize.
* `column_width=None`: `ColumnWidth` or `list[ColumnWidth]`: (Optional) column width.
* `filter_logic=None`: `string`: (Optional) filtering logic (`'and` / `or'`).
* `row_height=None`: `integer`: (Optional) row height in pixels.
* `sync_aggregation=None`: `bool`: (Optional) synchronize columns and metrics.

Values returned:

* `id`: `string`: Table ID.

</Cut>

<Cut title="create_metric_table_config(self, metrics, slices, diff_type, pvalue_thresholds)">

Creating a table with metric parameters.

* `metrics=None`: `list[string]`: (Optional) metric names.
* `slices=None`: `list[string]`: (Optional) slice names.
* `diff_type=None`: `string`: (Optional) type of comparison (`'Abs'` or `'Perc'`).
* `pvalue_thresholds=None`: `list[floats]`: (Optional) list of three floats with threshold p-values to indicate colors.

Values returned:

* `id`: `string`: Table ID.

</Cut>

## Sorting {#sorting}

A class of sorting parameters.

### Parameters {#parameters}

* `name`: `string`: Column name.
* `direction='asc'`: `string`: Sorting in ascending or descending order (`'asc'` or `'desc'`).
* `subindex=None`: `integer`: (Optional) subcolumn index.

## Filtering {#filtering}

A class of filtering parameters.

### Parameters {#parameters}

* `name`: `string`: Column name.
* `value`: `string`: Filter value.
* `subindex=None`: `integer`(Optional) subcolumn index.
* `operator=None`: `string`: Filtering operator (`'equal'`, `'notEqual'`, `'greaterThan'`, `'lessThan'`, `'greaterThanOrEqual'`, and `'lessThanOrEqual'`).

## ColumnWidth {#column-width}

A class of column width.

### Parameters {#parameters}

* `name`: `string`: Column name.
* `width`: `integer`: Column width in pixels.
* `subindex=0`: `integer`: (Optional) subcolumn index.
