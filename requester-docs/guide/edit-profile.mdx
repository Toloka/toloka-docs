# Editing the profile

## How to edit the profile

1. Go to the [requester profile](https://platform.toloka.ai/requester/profile).

1. Click <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1675425178780/1675425178780_original_svg.svg" alt="" /> → **Edit** next to the **Account info** label.

1. You can edit the following fields:

    - **LAST NAME**

    - **FIRST NAME**

    - **DISPLAY NAME**

    - **EMAIL**

    - **PHONE**

    <Note mode="warning" title="Warning">

    Please note that the display name is the name that the Tolokers see as the author of the tasks. It must be unique for every client.

    </Note>

1. Click **Save**.

## How to delete the profile

<Note mode="restriction" title="Alert">

You might lose the project data if you delete your requester account.

</Note>

If you want to delete your requester account:

1. Go to the [requester profile](https://platform.toloka.ai/requester/profile).

1. Click <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1675425178780/1675425178780_original_svg.svg" alt="" /> → **Delete** next to the **Account info** label.
