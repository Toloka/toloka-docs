# Receiving responses

You can download Tolokers' responses in a file as tasks are completed.

If tasks were distributed with an [overlap](/docs/glossary#overlap) of more than 2, run [aggregation](/docs/guide/result-aggregation). Toloka will process all Tolokers' responses for the task and form the resulting response.
