# Special components and helpers

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a task interface in [Template Builder](/docs/template-builder/reference/).

</Note>

To create a [task interface](/docs/glossary#task-interface), go to the [HTML](/docs/guide/spec#html) section on the [project](/docs/glossary#project) page. Here's what you can use:

- [Handlebars expressions and helpers](/docs/guide/t-components/handlebars) (see the [documentation on the Handlebars website](http://handlebarsjs.com/)).

- Additional helpers:

    - [Concatenation](/docs/guide/t-components/helpers#concat)
    - [Object operations](/docs/guide/t-components/helpers#object)
    - [Logical operations and comparisons](/docs/guide/t-components/helpers#equal)
    - [Recording output data field values in JSON](/docs/guide/t-components/helpers#js_fields)

- Components for solving common problems:

    - [String input field](/docs/guide/t-components/string)
    - [Text input field](/docs/guide/t-components/text)
    - [String input with suggest list](/docs/guide/t-components/suggest)
    - [Switch](/docs/guide/t-components/radiobuttons)
    - [Checkbox](/docs/guide/t-components/checkboxes)
    - [Dropdown list](/docs/guide/t-components/list)
    - [Button for file upload](/docs/guide/t-components/upload-file)
    - [Button for image upload](/docs/guide/t-components/upload-picture)
    - [Button for recording and uploading an audio file](/docs/guide/t-components/upload-audio)
    - [Button with click validation](/docs/guide/t-components/button-link)
    - [Image with area selection](/docs/guide/t-components/image-annotation)
    - [Text segmentation editor](/docs/guide/t-components/segmentation-editor)
    - [Image](/docs/guide/t-components/img)
    - [Button](/docs/guide/t-components/button)
    - [Iframe](/docs/guide/t-components/iframe)
    - [Text with HTML formatting](/docs/guide/t-components/html)

Libraries for using Handlebars and additional components are connected by default when you create a project.
