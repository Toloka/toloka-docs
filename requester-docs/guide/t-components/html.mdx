# Text formatted in HTML

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a task interface in [Template Builder](/docs/template-builder/).

</Note>

To upload a task with formatted text (HTML tags and styles):

1. Add a field of `string` type into the [input data description](/docs/guide/incoming). For example:

    ```json
    {
    "html": {
    "type": "string",
    "required": true
    }
    }
    ```

1. In the [task interface](/docs/guide/spec) put the name of the input data field in triple curly brackets. For example: `{{{html}}}`.

Before uploading a task, make sure that all quotation marks in the formatted text are [escaped](/docs/guide/pool_csv#string).
