# Iframe

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating an iframe in [Template Builder](/docs/template-builder/reference/view.iframe).

</Note>

To add an inline frame to a task, use the `{{iframe src=<object URL>}}` component. Example:

```plaintext
{{iframe src=url width="100px" height="100px"}}
```

## Parameters

**Parameter**|**Description**|**Required**|**Default value**
----- | ----- | ----- | -----
`src`| Page URL. | yes | no
`width`| <IframeTable1/>| no | `"300px"`
`height`| <IframeTable2/>| no | `"500px"`
`content-width`| The width of the content in the frame, in pixels. For example, `content-width="700`".<br/><br/>To make the frame the width of the screen, set `content-width="false".` | no | `"1280"`
`zoom`| <IframeTable3/>| no | `false`
`class`| CSS class for the frame. | no | `".iframe"`
`sandbox`| <IframeTable4/>| no | `"allow-scripts allow-same-origin"`
