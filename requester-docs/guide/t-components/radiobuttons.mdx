# Radio button

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a radio button in [Template Builder](/docs/template-builder/reference/field.radio-group).

</Note>

To add radio buttons to the task, use the `{{field type="radio" name="<output field name>"}}` component. For example, a set of two radio buttons:

```plaintext
{{field type="radio" name="result" label="Yes" value="Yes" hotkey="1"}}
{{field type="radio" name="result" label="No" value="No" hotkey="2"}}
```

Add a field of `string` type in the [output data description](/docs/guide/incoming). Example:

```json
{
  "result": {
    "type": "string",
    "required": true
  }
}
```
