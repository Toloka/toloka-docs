# Button

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a button in [Template Builder](/docs/template-builder/reference/field.button-radio).

</Note>

To add a button to a task, use the `{{button}}` component. Example:

```plaintext
{{button label="Click me" href="https://yandex.ru" action=true}}
```
