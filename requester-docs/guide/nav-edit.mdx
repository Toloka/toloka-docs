# Editing a skill

## Editing the skill parameters `{#reward-edit}`

1. Open the **Skills** page.

1. Hover the cursor over the desired skill and click <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1675425172884/1675425172884_original_svg.svg" alt="" />.

1. Change the skill name, type, or description and click **Save**.
