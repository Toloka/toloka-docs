## Add files to the instructions `{#instruction}`
<Note mode="warning" title="Warning">

Set the access rights for the folder with instructions to “Public”.

</Note>

To add an image to the [project instructions](/docs/guide/instruction):

1. Upload files to the `Applications/Toloka/<folder name>` folder using the [web interface](https://disk.yandex.com/client/disk) or [Yandex Disk](https://disk.yandex.com/promo/download) app.

## Frequently asked questions `{#faq}`

<Cut title="I am a new client. Can I connect Yandex Disk?">

Unfortunately, no. As Toloka no longer supports Yandex Disk integration for new users, you can't connect it. Please use one of the photo hosting services: [wampi](https://wampi.ru/?lang=en), [imgbb](https://imgbb.com/), [ImageShack](https://imageshack.com/), or [imgur](https://imgur.com/).

</Cut>

<Cut title="I have a Yandex Disk integration, but want to manage the proxy. How can I do that?">

Please [contact our support team](/docs/guide/troubleshooting/support?form-topic1=other&form-source=guide-prepare-data) for any actions with Yandex Disk proxy.

</Cut>

<Cut title="I have a Yandex Disk integration, but want to disconnect it. How can I do that?">

</Cut>
