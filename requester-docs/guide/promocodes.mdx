# Promo codes

To activate a promo code:

1. On the [Profile](https://platform.toloka.ai/requester/profile) page, click **Enter promo code**.

    <Image width="640px" withBackground imageId="1675426731867" alt="" />

1. Enter your promo code and click **Activate**.

    <Image width="435px" withBackground imageId="1675426728205" alt="" />

1. Done! Promo code activated

<Note mode="warning" title="Warning">

The same promo code can't be activated twice.

</Note>
