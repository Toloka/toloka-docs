# Adding tasks to a pool

To add tasks to a pool:

1. [Place the files for tasks](/docs/guide/cloud-storage).

1. [Create a file with tasks](/docs/guide/pool_csv).

1. [Upload tasks to the pool](/docs/guide/task_upload).

    <Note mode="restriction" title="Restriction">

    You can add up to one million tasks to the pool. To upload more tasks, create another pool.

    </Note>

1. If you haven't yet marked up [control](/docs/glossary#control-task) and [training](/docs/glossary#training-task) tasks in the file, [mark up the tasks in the interface](/docs/guide/task_markup).

    <Note mode="note" title="Note">

    If you created a project in the [sandbox](/docs/glossary#sandbox), keep in mind that when you move a project to the [Toloka production version](https://platform.toloka.ai/requester/), tasks aren't moved, including those that are already labeled.

    </Note>
