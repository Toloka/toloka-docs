# Creating a project

If you already have a project and you want to create an identical one, [clone](#clone) the existing project. If not, create a new project.

## How to create a project {#new-project}

The project defines what the task will look like for a Toloker.

To get better results, make tasks as simple as possible and divide complex tasks into several projects.

<Note mode="note" title="Note">

You can first create and test your project for free in the [sandbox](/docs/guide/sandbox) and then [move](/docs/guide/sandbox#export) it to the Toloka production version.

</Note>

To create a project, follow the instructions:

#### In the interface:

1. Choose a preset:

    1. Click the **Create project** button on the **Projects** page.

    1. Choose a preset. The preset contains pre-configured [input and output data fields](/docs/guide/incoming) and [the task interface](/docs/guide/spec), which you can edit.

        If there is no suitable preset, choose an empty preset.

        <Note mode="tip" title="Tip">

        If you unsure which preset to choose, you can ask Toloka experts to create a project for you. To do that,

        - click **Get expert help**,
        - select the way Toloka can contact you (WhatsApp, Telegram, Viber, email, or phone),
        - enter your phone number or email address,
        - briefly describe the project and the results you want to get.

        Toloka engineering team will contact you for more details and explain your further steps.

        </Note>

1. Provide general information:

    1. Enter the **Name** and **Description**. Tolokers will see this in the task list.

    1. Optionally add a **Private comment**.

1. Edit the task interface:

    <Tabs tabs={[
      { title: 'Template Builder', value: <ProjectList1/> },
      { title: 'HTML/JS/CSS editor', value: <ProjectList2/> },
    ]}/>

1. [Write guidelines](/docs/guide/instruction) for Tolokers.

    You can prepare instructions in HTML format, then copy and paste into the editor. Click **&lt;>** to switch to HTML mode.

1. Optionally, add translations to other languages.

    1. Select the source language and the target languages.

    1. Fill in the fields in the table.

    1. Click **Finish**.

After creating the project, add a [task pool](/docs/guide/pool-main) to it. You can also set up [quality control](/docs/guide/control) in the project.

## How to clone a project {#clone}

You can view the list of created projects on the **Projects** page. Open the project page to view the list of pools, [project rating](/docs/guide/project_rating_stat), and [project statistics](/docs/guide/project-statistic).

To clone a project, click <img src="https://tlk-infra-front.azureedge.net/portal-static/uploaded/images/1675425192399/1675425192399_original_svg.svg" alt="" /> in the list of projects or select **Project actions â†’ Clone** on the project page.

<Note mode="warning" title="Warning">

Quality control in the project is not cloned.

</Note>

If you need to change the project settings, [open edit mode](/docs/guide/edit-project).
