# Getting API key

Before you start using Toloka-Kit library, register in either the production or sandbox Toloka web version, get your key for the API access, and top up your account.

## Registration

1. [Register](/docs/guide/access) in Toloka as a requester.

1. Toloka has two versions:

    - The [production](https://platform.toloka.ai/auth?role=REQUESTER) version is used by default in [this example](/docs/toloka-kit/recipes/learn-basics).

    - The [sandbox](https://platform.sandbox.toloka.ai/auth?role=REQUESTER) version is a testing environment for Toloka. [Learn more](/docs/guide/sandbox)

## Getting API key `{#oauth-token}`

Get an API key for your version. Go to [Profile → Integrations](https://platform.toloka.ai/requester/profile/integration):

<Image width="700px" withBackground imageId="e12badf3c4c7296d99750b5ba887635a" alt="List of keys" />

Refer to the [Generating API keys](/docs/guide/api-token) section to learn how to issue API keys.

## Top up account `{#top-up}`

If you're going to use the production version, [add funds](/docs/guide/refill) to your Toloka account.
