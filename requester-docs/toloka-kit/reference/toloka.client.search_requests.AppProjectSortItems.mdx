# AppProjectSortItems

`toloka.client.search_requests.AppProjectSortItems`

```python
AppProjectSortItems(self, items=None)
```

Keys for sorting App projects in search results.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`items`|**Optional\[List\[[SortItem](/docs/toloka-kit/reference/toloka.client.search_requests.AppProjectSortItems.SortItem)\]\]**|<p>A list of sorting keys. Supported values:</p> <ul> <li>`'id'` — An App project ID.</li> <li>`'name'` — An App project name.</li> <li>`'created'` — A project creation date.</li> </ul>
