# set_user_restriction

`toloka.client.TolokaClient.set_user_restriction` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.2.1/src/client/__init__.py#L3378)

```python
set_user_restriction(self, user_restriction: UserRestriction)
```

Restricts access to projects or pools for a Toloker.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`user_restriction`|**[UserRestriction](/docs/toloka-kit/reference/toloka.client.user_restriction.UserRestriction)**|<p>Restriction parameters.</p>

* **Returns:**

  Updated restriction object.

* **Return type:**

  [UserRestriction](/docs/toloka-kit/reference/toloka.client.user_restriction.UserRestriction)

**Examples:**

Restricting access to a project.

```python
new_restriction = toloka_client.set_user_restriction(
    toloka.client.user_restriction.ProjectUserRestriction(
        user_id='fac97860c7929add8048ed2ef63b66fd',
        private_comment='The Toloker often makes mistakes',
        project_id='92694'
    )
)
```
