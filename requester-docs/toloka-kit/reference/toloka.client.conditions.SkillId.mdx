# SkillId

The ID of a changed skill which caused access blocking.

`SkillId` provides details if the [PoolAccessRevokedReason](/docs/toloka-kit/reference/toloka.client.conditions.PoolAccessRevokedReason) condition equals `SKILL_CHANGE`.

`SkillId` is used with collectors:
- [UsersAssessment](/docs/toloka-kit/reference/toloka.client.collectors.UsersAssessment)
