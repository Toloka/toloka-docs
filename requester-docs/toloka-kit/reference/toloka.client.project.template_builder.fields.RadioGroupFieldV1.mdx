A component for selecting one value out of several options.

For more information, see [field.radio-group](/docs/template-builder/reference/field.radio-group).

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`data`|**Optional\[[BaseComponent](/docs/toloka-kit/reference/toloka.client.project.template_builder.base.BaseComponent)\]**|<p>A data path.</p>
`options`|**Optional\[Union\[[BaseComponent](/docs/toloka-kit/reference/toloka.client.project.template_builder.base.BaseComponent), List\[Union\[[BaseComponent](/docs/toloka-kit/reference/toloka.client.project.template_builder.base.BaseComponent), [GroupFieldOption](/docs/toloka-kit/reference/toloka.client.project.template_builder.fields.GroupFieldOption)\]\]\]\]**|<p>A list of options.</p>
`disabled`|**Optional\[Union\[[BaseComponent](/docs/toloka-kit/reference/toloka.client.project.template_builder.base.BaseComponent), bool\]\]**|<p>Disabling the component:</p> <ul> <li>`False` — Selecting an option is allowed.</li> <li>`True` — Selecting an option is disabled.</li> </ul> <p>Default value: `False`.</p>
`hint`|**Optional\[Any\]**|<p>A hint.</p>

```python
radio_group_field = tb.fields.RadioGroupFieldV1(
```
