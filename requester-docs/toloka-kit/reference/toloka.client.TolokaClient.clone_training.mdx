# clone_training

`toloka.client.TolokaClient.clone_training` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.2.0/src/client/__init__.py#L1900)

```python
clone_training(self, training_id: str)
```

Clones an existing training.

An empty training with the same parameters is created.
The new training is attached to the same project.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`training_id`|**str**|<p>The ID of the training to be cloned.</p>

* **Returns:**

  The new training.

* **Return type:**

  [Training](/docs/toloka-kit/reference/toloka.client.training.Training)

**Examples:**

```python
toloka_client.clone_training(training_id='1')
```
