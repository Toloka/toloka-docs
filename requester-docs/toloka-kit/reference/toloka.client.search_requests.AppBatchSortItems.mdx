# AppBatchSortItems
`toloka.client.search_requests.AppBatchSortItems`

```python
AppBatchSortItems(self, items=None)
```

Keys for sorting App batches in search results.

| Parameters | Type | Description |
| :----------| :----| :-----------|
`items`|**Optional\[List\[[SortItem](/docs/toloka-kit/reference/toloka.client.search_requests.AppBatchSortItems.SortItem)\]\]**|<p>A list of sorting keys. Supported values:</p> <ul> <li>`&#x27;id&#x27;` — A batch ID.</li> <li>`&#x27;name&#x27;` — A batch name.</li> <li>`&#x27;created&#x27;` — A batch creation date.</li> <li>`&#x27;status&#x27;` — The item status.</li> </ul>
