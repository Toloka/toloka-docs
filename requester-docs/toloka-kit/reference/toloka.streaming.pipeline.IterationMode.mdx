# IterationMode

`toloka.streaming.pipeline.IterationMode` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.2.0/src/streaming/pipeline.py#L70)

Possible values:

* `ALL_COMPLETED` – start next iteration only when all current tasks are done.
* `FIRST_COMPLETED` – start next iteration as soon as any single task is done.
