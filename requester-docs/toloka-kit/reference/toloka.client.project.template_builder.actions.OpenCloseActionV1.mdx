The action changes the display mode of another component.

It can expand an [image](/docs/toloka-kit/reference/toloka.client.project.template_builder.view.ImageViewV1) to a full screen
or collapse a [section](/docs/toloka-kit/reference/toloka.client.project.template_builder.view.CollapseViewV1).

For more information, see [action.open-close](/docs/template-builder/reference/action.open-close).
