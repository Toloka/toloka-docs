# MessageThreadSortItems

`toloka.client.search_requests.MessageThreadSortItems`

```python
MessageThreadSortItems(self, items=None)
```

Keys for sorting message threads in search results.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`items`|**Optional\[List\[[SortItem](/docs/toloka-kit/reference/toloka.client.search_requests.MessageThreadSortItems.SortItem)\]\]**|<p>A list of sorting keys. Supported values:</p> <ul> <li>`'id'` — Message thread ID.</li> <li>`'created'` — Message thread creation date.</li> </ul>
