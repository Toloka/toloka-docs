## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`id`|**Optional\[str\]**|<p>The ID of the Toloker's skill value.</p>

`modified`|**Optional\[datetime\]**|<p>The date and time when the skill value was updated.</p>
