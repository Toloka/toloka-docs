# UserBonusBatchCreateResult

`toloka.client.batch_create_results.UserBonusBatchCreateResult`

```python
UserBonusBatchCreateResult(
    self,
    *,
    items: Optional[Dict[str, UserBonus]] = None,
    validation_errors: Optional[Dict[str, Dict[str, FieldValidationError]]] = None
)
```

The result of issuing bonuses for Tolokers.

`UserBonusBatchCreateResult` is returned by the [create_user_bonuses](/docs/toloka-kit/reference/toloka.client.TolokaClient.create_user_bonuses) method.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`items`|**Optional\[Dict\[str, [UserBonus](/docs/toloka-kit/reference/toloka.client.user_bonus.UserBonus)\]\]**|<p>A dictionary with created bonuses. The indexes of a `create_user_bonuses` input list are used as keys in the dictionary.</p>
`validation_errors`|**Optional\[Dict\[str, Dict\[str, [FieldValidationError](/docs/toloka-kit/reference/toloka.client.batch_create_results.FieldValidationError)\]\]\]**|<p>A dictionary with validation errors. It is filled if the request parameter `skip_invalid_items` is `True`.</p>
