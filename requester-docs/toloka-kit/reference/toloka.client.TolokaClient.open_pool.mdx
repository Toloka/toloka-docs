# open_pool
`toloka.client.TolokaClient.open_pool` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.1.4/src/client/__init__.py#L1622)

Opens a pool.

After opening the pool, tasks can be assigned to Tolokers.

## Parameters Description

| Parameters | Type | Description |
| :----------| :----| :-----------|

* **Returns:**

  The pool with updated status.

* **Return type:**

  [Pool](/docs/toloka-kit/reference/toloka.client.pool.Pool)

Opening a pool.

