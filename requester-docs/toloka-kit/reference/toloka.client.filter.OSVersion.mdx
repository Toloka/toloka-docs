# OSVersion

`toloka.client.filter.OSVersion` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.2.0/src/client/filter.py#L597)

```python
OSVersion(
    self,
    operator: CompareOperator,
    value: float
)
```

Filtering Tolokers by an OS version.

The version consists of major and minor version numbers, for example, `14.4`.
The version is represented as a single floating point number in conditions.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`operator`|**[CompareOperator](/docs/toloka-kit/reference/toloka.client.primitives.operators.CompareOperator)**|<p>An operator used in a condition. Allowed set of operators depends on the filter.</p>
`value`|**float**|<p>The version of the OS.</p>
