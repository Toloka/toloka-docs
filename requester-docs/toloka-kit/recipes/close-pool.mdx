# Close pool

_Close an opened pool in Toloka._

## Steps to follow

<Import/>

<Instantiate/>

### 3. Close pool {#step-three}

[Find out](/docs/toloka-kit/recipes/get-pools) the ID of the pool you want to close. Then call the `close_pool()` method.

<CodeCopy
  isSmall
  theme="light"
  code={
    <HighlightChildren language="python">
      closed_pool = toloka_client.<Term term="close_pool"><ClosePool/></Term>('1443858')

  }

/>

### 4. Print pool ID and status {#step-four}

The `close_pool()` request will return the [Pool](/docs/toloka-kit/reference/toloka.client.pool.Pool) class object. You can use its attributes to print the information you need.

```python
print(closed_pool.id, closed_pool.status)
```

You should get an output with the pool ID and the updated status which looks like this.

```bash
1443858 Status.CLOSED
```

## Complete code: Close pool {#complete-code}

```python
import toloka.client as toloka

toloka_client = toloka.TolokaClient('AQC2AGAJgyNSA8CtpdO9MWy_QEB6s6kDjHUoElE', 'PRODUCTION')

closed_pool = toloka_client.close_pool('1443858')
print(closed_pool.id, closed_pool.status)
```
