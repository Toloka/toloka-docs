# Archive project

_Move the project which you no longer use into archive._

## Steps to follow

<Import/>

<Instantiate/>

### 3. Archive project {#step-three}

[Find out](/docs/toloka-kit/recipes/get-projects) the ID of the project which you want to archive. Then call the `archive_project()` method.

<CodeCopy
  isSmall
  theme="light"
  code={
    <HighlightChildren language="python">
      archived_project = toloka_client.<Term term="archive_project"><ArchiveProject/></Term>('117493')
    </HighlightChildren>
  }
  textToCopy={`archived_project = toloka_client.archive_project('117493')`}
/>

### 4. Print project ID and status {#step-four}

The `archive_project()` request will return the [Project](/docs/toloka-kit/reference/toloka.client.project.Project) class object. You can use its attributes to print the information you need.

```python
print(archived_project.id, archived_project.status)
```

You should get an output with the project ID and the updated status which looks like this.

```bash
117493 ProjectStatus.ARCHIVED
```

## Complete code: Archive project {#complete-code}

```python
import toloka.client as toloka

toloka_client = toloka.TolokaClient('PlaceYourRealOAuthToken_Here', 'PRODUCTION')

archived_project = toloka_client.archive_project('117493')
print(archived_project.id, archived_project.status)
```
