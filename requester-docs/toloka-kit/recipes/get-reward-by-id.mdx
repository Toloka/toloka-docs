# Get bonus details

_Get the detailed information about the bonus with the ID specified in the request._

## Steps to follow

<Import/>

<Instantiate/>

### 3. Get bonus information {#step-three}

[Find out](/docs/toloka-kit/recipes/get-rewards) the ID of the bonus for which you want to get the detailed information. Then get this info calling the `get_user_bonus()` method.

<CodeCopy
  isSmall
  theme="light"

      bonus = toloka_client.<Term term="get_user_bonus"><GetUserBonus/></Term>('3139')

### 4. Print bonus title and amount {#step-four}

The `get_user_bonus()` request will return the [UserBonus](/docs/toloka-kit/reference/toloka.client.user_bonus.UserBonus) class object. You can use its attributes to print the information you need.

```python
print(bonus.public_title['EN'], bonus.amount)
```

You should get an output with the bonus title and amount which looks like this.

```bash
Perfect job! 0.500
```

## Complete code: Get bonus details {#complete-code}

```python
import toloka.client as toloka

toloka_client = toloka.TolokaClient('PlaceYourRealOAuthToken_Here', 'PRODUCTION')

bonus = toloka_client.get_user_bonus('3139')
print(bonus.public_title['EN'], bonus.amount)
```
