# Change pool priority

_Change the priority of the pool and offer it to Tolokers before other pools._

## Steps to follow

<Import/>

<Instantiate/>

### 3. Change pool priority `{#step-three}`

[Find out](/docs/toloka-kit/recipes/get-pools) the ID of the pool for which you want to change the priority. Then call the `patch_pool()` method.

<CodeCopy
  isSmall
  theme="light"
  code={
    <HighlightChildren language="python">
      patched_pool = toloka_client.<Term term="patch_pool"><PatchPool/></Term>(pool_id='1443815', priority=0)
    </HighlightChildren>
  }
  textToCopy={`patched_pool = toloka_client.patch_pool(pool_id='1443815', priority=0)`}
/>

### 4. Print pool ID and priority `{#step-four}`

The `patch_pool()` request will return the [Pool](/docs/toloka-kit/reference/toloka.client.pool.Pool) class object. You can use its attributes to print the information you need.

```python
print(patched_pool.id, patched_pool.priority)
```

You should get an output with the pool ID and the updated priority which looks like this.

```bash
1443815 0
```

## Complete code: Change pool priority `{#complete-code}`

```python
import toloka.client as toloka

toloka_client = toloka.TolokaClient('PlaceYourRealOAuthToken_Here', 'PRODUCTION')

patched_pool = toloka_client.patch_pool(pool_id='1443815', priority=0)
print(patched_pool.id, patched_pool.priority)
```
