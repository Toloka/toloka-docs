# Accessing the API

To send an authorization request to the API, you need an API key.

**Query example**

```http
GET /api/app/v0/apps
Host: https://toloka.dev
Authorization: ApiKey PlaceYourRealApiKey_Here
```

## Getting an API key

- [Register](/docs/guide/access) in Toloka.

- Get an API key in the [requester interface](https://platform.toloka.ai/requester/profile/integration).
