# Add items in bulk

## Description

Adds multiple items to a batch in the project with the ID specified in the request, with the maximum of 10,000 items per request.

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/apps-reference/#post-/app-projects/-app_project_id-/items/bulk):** more parameter details, request/response examples, and code samples in various programming languages.

</Note>

## Request

method | url
----- | -----

### Parameters

name | type | required | description
----- | ----- | ----- | -----
app_project_id | path | true | Project ID.
