# Toloka LLM workflow overview

When you register in Toloka LLM, the platform creates the first project automatically. You will see the project interface when you sign in.

To start working with it,

- [upload a dataset](/docs/toloka-llm/dataset),
- [add classes](/docs/toloka-llm/classes) and their descriptions (in case your uploaded dataset misses them),
- [label several items](/docs/toloka-llm/label) and check if the model is correctly set up and works as expected,
- [describe your task](/docs/toloka-llm/task-description),
- [check the generated prompt](/docs/toloka-llm/prompt) and enhance it if needed or experiment with other prompts,
- run tests to [produce better variants](/docs/toloka-llm/variants).

<Note mode="note" title="Note">

Most of the steps above are optional, but we recommend that you don't skip them for better results.

</Note>

After that you can [deploy the project variants](/docs/toloka-llm/deploy) to try it on a real dataset.

## Managing projects

To manage your projects:

1. Click <img src="https://toloka.ai/files/edit.svg" alt="Edit" /> in the upper-left header area next to the project name.

1. In the list of **My projects** find the project name you want to edit.

    - Click the project name to switch to it.

    - Click <img src="https://toloka.ai/files/edit.svg" alt="Edit project name" /> next to the project name to change it, then <img src="https://toloka.ai/files/done.svg" alt="Accept changes" /> to accept changes.

    - Click <img src="https://toloka.ai/files/delete.svg" alt="Delete project" /> next to the project name to delete the project.

    - Click **+ Create a project** to create another project and switch to it.
