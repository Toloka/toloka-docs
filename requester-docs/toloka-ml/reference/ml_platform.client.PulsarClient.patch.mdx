# patch

`ml_platform.client.PulsarClient.patch`

```python
patch(
    self,
    run_ids: Union[str, List[str]],
    model: Optional[ModelInfo] = None,
    dataset: Optional[DatasetInfo] = None,
    result: Optional[Dict] = None,
    tags: Optional[List[str]] = None,
    user_datetime: Optional[datetime] = None,
    name: Optional[str] = None,
    description: Optional[str] = None,
    model_output_url: Optional[str] = None,
    per_object_data_metainfo: Optional[List[Dict]] = None,
    diff_tool_config: Optional[Dict] = None,
    report_url: Optional[str] = None,
    permissions: Optional[Permissions] = None,
    produced_model_uid: Optional[str] = None,
    produced_dataset_uid: Optional[str] = None,
    links: Optional[List[str]] = None,
    metrics: Optional[Metrics] = None,
    experiment_id: Optional[str] = None,
    status: Optional[str] = None,
    metrics_ttl: Optional[str] = None,
    user_data: Optional[Dict] = None,
    metadata: Optional[Dict] = None
)
```

Updates specified properties of run.
