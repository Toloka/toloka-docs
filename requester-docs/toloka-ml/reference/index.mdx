# Python API reference

The API lets you use Toloka ML platform tools based on Python methods.

<PipInstall/>

## Get OAuth token {#token}

Toloka ML Python client requires OAuth token to send requests, launch experiments, monitor the status of running graphs, and display a model's output and per object diffs.

To get your token, [sign in](/docs/toloka-ml/quickstart#authorization) to the platform via the platform interface and click [this link](https://tolokamodels.tech/token).
