# ExperimentsClient

`ml_platform.experiments.ExperimentsClient`

```python
ExperimentsClient(
    self,
    token: Optional[str] = None,
    http_client: Optional[HttpClient] = None
)
```

Provides API methods for working with experiments.

## Methods summary

| Method | Description |
| :------| :-----------|
[add](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.add)| Submits the simple experiment.
[add_runs](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.add_runs)| Adds runs to the experiment by IDs.
[delete](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.delete)| Deletes the experiment.
[find](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.find)| Finds experiments.
[get_best_run](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.get_best_run)| Gets the best run from the experiment by ID.
[patch](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.patch)| Patches the experiment.
[remove_runs](/docs/toloka-ml/reference/ml_platform.experiments.ExperimentsClient.remove_runs)| Removes runs from the experiment by IDs.
