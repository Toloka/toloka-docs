# update

`ml_platform.datasets.DatasetsClient.update`

```python
update(self, dataset: Dataset)
```

Overwrites all properties of the dataset or the dataset version without new version creation.

You cannot change dataset files with such an update.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`dataset`|**[Dataset](/docs/toloka-ml/reference/ml_platform.datasets.Dataset)**|<p>Dataset with the changed fields.</p>
