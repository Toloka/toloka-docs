# get_sliced_aggregation_diff

`ml_platform.diff_tool.DiffTool.get_sliced_aggregation_diff`

```python
get_sliced_aggregation_diff(
    self,
    baseline_id: str,
    competitor_ids: Union[str, List[str]],
    slice_columns: List[str],
    metric_columns: Optional[List[str]] = None,
    group_ids: Optional[List[str]] = None,
    replace_invalid_json_values: bool = False,
    pvalue_threshold: Optional[float] = None,
    stat_tests: Optional[List[str]] = None
)
```

Builds object sliced aggregation diff.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`baseline_id`|**str**|<p>Baseline run ID.</p>
`competitor_ids`|**Union\[str, List\[str\]\]**|<p>Competitor run IDs.</p>
`slice_columns`|**List\[str\]**|<p>Columns to take slices.</p>
`metric_columns`|**Optional\[List\[str\]\]**|<p>Columns to calculate diff.</p>
`group_ids`|**Optional\[List\[str\]\]**|<p>Group ids of metrics columns to calculate diff. Columns with `group_id` from `group_ids` will be added to metrics columns.</p>
`replace_invalid_json_values`|**bool**|<p>Whether to replace invalid JSON values from response. If `True`, float `nans` and `infs` will be replaced with `None`.</p>
`pvalue_threshold`|**Optional\[float\]**|<p>P-value threshold for the returned slices. All diffs with p-value &gt; `pvalue_threshold` will be skipped.</p>
`stat_tests`|**Optional\[List\[str\]\]**|<p>Stat test to calculate `pvalue`. Should be from `{Wilcoxon, Ttest}`.</p>

* **Returns:**

  Sliced aggregation diff.

* **Return type:**

  Dict
