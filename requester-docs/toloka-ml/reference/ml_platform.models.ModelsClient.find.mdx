# find

`ml_platform.models.ModelsClient.find`

```python
find(
    self,
    uids: Optional[List[str]] = None,
    names: Optional[List[str]] = None,
    versions: Optional[List[str]] = None,
    usernames: Optional[List[str]] = None,
    model_types: Optional[List[str]] = None,
    statuses: Optional[List[str]] = None,
    tags: Optional[List[str]] = None,
    date_from: Optional[str] = None,
    date_to: Optional[str] = None,
    offset: int = 0,
    limit: int = 1000
)
```

Finds a model.

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`uids`|**Optional\[List\[str\]\]**|<p>Model IDs.</p>
`names`|**Optional\[List\[str\]\]**|<p>Model names.</p>
`versions`|**Optional\[List\[str\]\]**|<p>Model versions.</p>
`usernames`|**Optional\[List\[str\]\]**|<p>Creator usernames.</p>
`model_types`|**Optional\[List\[str\]\]**|<p>Model types.</p>
`statuses`|**Optional\[List\[str\]\]**|<p>Model statuses.</p>
`tags`|**Optional\[List\[str\]\]**|<p>Model tags.</p>
`date_from`|**Optional\[str\]**|<p>Start date to search for models from.</p>
`date_to`|**Optional\[str\]**|<p>End date to search for models to.</p>
`offset`|**int**|<p>The number of models to skip.</p>
`limit`|**int**|<p>The maximum number of models to return.</p>

* **Returns:**

  List of models.
