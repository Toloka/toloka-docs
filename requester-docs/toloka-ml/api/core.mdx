# Main Structures

Basic structures for working with the Toloka ML platform API.

## Permissions {#permissions}

A structure for granting access rights.

### Parameters {#parameters}

* `read`: `list[string]`: (Optional) list of groups with read permissions.
* `write`: `list[string]`: (Optional) list of groups with `read` and edit permissions.
* `owner`: `list[string]`: List of groups of owners who can change access rights and have the same permissions as `read` and `write`.

## Metric {#metric}

Metric class.

### Parameters {#parameters}

* `name`: `string`: Metric name.
* `value`: `any`: Metric value.
* `slice=None`: `string`: (Optional) slice name.
* `best_value=None`: `string`: (Optional) best metric value.
* `type=None`: `string`: (Optional) metric type.
* `tags=None`: `list[string]`: (Optional) tags.
* `main=None`: `bool`: (Optional) main metric.

## Metrics {#metrics}

A structure for getting metrics.

### Parameters {#parameters}

* `metrics`: `list[Metric]`: List of metrics.

## RunInfo {#run-info}

A structure with run information.

### Parameters {#parameters}

* `model`: `ModelInfo`: Model information.
* `dataset`: `DatasetInfo`: Dataset information.
* `result=None`: `map`: (Optional) run metric values.
* `tags=None`: `list[string]`: (Optional) tags.
* `user_datetime=None`: `datetime`: (Optional) user date.
* `name=None`: `string`: (Optional) run name.
* `description=None`: `string`: (Optional) run description.
* `per_object_data_metainfo=None`: `list`: (Optional) Model results per object data.
* `diff_tool_config=None`: `dict`: (Optional) Diff Tool configuration.
* `report_url=None`: `string`: (Optional) report URL.
* `permissions=None`: `Permissions`: (Optional) access rights.
* `produced_model_uid=None`: `string`: (Optional) derived model's ID.
* `metrics_ttl=None`: `integer`: (Optional) metric time-to-live in days.
* `links=None:` `list`: (Optional) links to external services.
* `metrics=None`: `list[dict]`: (Optional) run metrics in glossary format:

* `dashboard_id=None`: `string`: (Optional) dashboard ID in Yandex Datalens.

## ParameterTuningConfig {#parameter-tuning-config}

Parameter selection settings.

### Parameters {#parameters}

* `workflow`: `WorkflowInfo`: Workflow whose parameters need to be optimized.
* `algorithm`: `ETuningAlgorithm`: Selection algorithm (`GridSearch`, `RandomSearch`, or `BayesOpt`).
* `metric`: `string`: Name of the metric to be optimized.
* `goal=EOptimizationGoal.Maximize`: `EOptimizationGoal`: (Optional) minimize or maximize the metric.
* `max_iterations=None`: `integer`: (Optional) maximum number of workflow runs.
* `search_space=None`: `SearchSpace`: (Optional) parameter search space.
* `random_seed=None`: `integer`: (Optional) random seed for parameter combinations.
* `warmup_iterations=None`: `integer`: (Optional) number of random iterations before starting Bayesian optimization.
