To delete the dataset,

1. Go to the [Datasets](https://tolokamodels.tech/datasets) section.

1. In the table with the dataset list, click the row of the dataset you want to delete.

1. On the dataset page, click **<img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1678438447502/1678438447502_original_svg.svg" alt="Delete dataset" /> Delete** next to the dataset name.

You will see a warning. Choose **<img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1678438447502/1678438447502_original_svg.svg" alt="Delete dataset" /> Delete permanently** to delete the dataset or **Cancel** to abort the action.

<Note mode="restriction" title="Alert">

If you delete the dataset, you will not be able to restore it later. You can only create it again.

</Note>
