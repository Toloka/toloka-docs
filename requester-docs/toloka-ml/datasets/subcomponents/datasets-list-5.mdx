After the dataset is created, it's placed in the [Toloka ML platform interface](https://tolokamodels.tech/) under **Datasets**. To view the dataset details, click the icon in the rightmost **Actions** column in the dataset row.

You can also open the dataset via the link

```http
https://tolokamodels.tech/datasets/dataset_id?dataset_version
```

If you do not specify the `dataset_version` in the link above, you will go to the latest available dataset version.

To view information about another version of the dataset, select it from the **Version** drop-down list.
