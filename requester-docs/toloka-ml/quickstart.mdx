# Quick start

This quick start will help you sign in to Toloka ML platform and start using it.

## Sign in {#authorization}

To work with Toloka ML platform, you must first sign in to it. To do this:

1. Go to the [Toloka ML platform interface](https://tolokamodels.tech/).

1. Click **Sign in** in the upper-right corner.

1. Choose the authentication method (currently, Toloka ML platform supports logging in via Google and GitHub) and follow the selected service authentication steps.

After you return to the Toloka ML platform interface, you will see the email you signed in with in the upper-right corner. To log out of the platform, hover the mouse cursor over it and choose **Log out**.

## Start using Toloka ML platform

Now you can start working with Toloka ML platform. To do this:

1. [Upload](/docs/toloka-ml/datasets#create) your first dataset to the platform and label it manually.

2. [Select a model](/docs/toloka-ml/models/collections) and use your dataset with it.

<Note mode="note" title="Note">

If you prefer to interact with Toloka ML platform using the Python library, then install the library, and get an OAuth token using the platform interface. After that, you can send requests and receive responses. Follow the [steps here](/docs/toloka-ml/reference/) to do this.

</Note>
