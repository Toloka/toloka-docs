# Datasets

A dataset is an entity that contains metadata and links to data. Datasets support versioning and access rights setup.

When adding, deleting, and updating files in a dataset, a new version is created. This lets you work with multiple versions of the same file and save information about them if they are deleted and updated.

## Setting up access permissions `{#permissions}`
Datasets have the following access levels:

* `read`: Getting information about a dataset and using its files.
* `write`: Includes the `read` permission and enables updating a dataset and creating new versions.
* `owner`: Includes the `write` permission and enables updating access rights. By default, the dataset owner is the user who created it.
