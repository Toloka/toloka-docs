# Metrics

Metrics are model quality criteria and one of the key attributes of a run. You can sort runs by metrics, compare them using the Toloka ML platform tools, and plot charts of multiple runs by time.

## Adding Main metrics to the table with runs {#filter}

To add Main metrics to the table with runs:

1. Open the [Toloka ML platform](https://tolokamodels.tech/) homepage.
2. Click ![manage columns](/_images/columns.png =26x16) in the table's workspace panel.

## Сравнение метрик {#diff}

Основное предназначение метрик — это сравнение показателей обучения. Toloka ML platform позволяет строить Diff предпосчитанных метрик, интегральных значений метрик и результатах семплирования.

Чтобы сравнить несколько runs по метрикам:

1. Откройте главную страницу [Toloka ML platform](https://tolokamodels.tech/).
2. Выберите runs при помощи опции слева от их идентификатора.
3. В появившемся меню слева снизу нажмите **Diff**. Произойдет переход на страницу [Diff Tool](https://tolokamodels.tech/diff), где можно будет сравнить runs.

В разделе **Metrics** указаны метрики, разбитые по названиям и срезам.

## Tensorboards HParams {#hparams}
