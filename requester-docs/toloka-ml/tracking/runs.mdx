# Runs

In Toloka ML platform every time you start a scenario to [train or apply the model](/docs/toloka-ml/models/collections/#model-actions), a separate run is created. You can combine multiple runs into a single group called an experiment.

<RunParameters/>

## Creating a run `{#add-run}`

When a run is created, a model name, a dataset name, and a set of metric values are saved to the Toloka ML platform database. In addition, you can set model and dataset options that can be viewed in the Toloka ML platform interface.
