<Cut title="Run and experiment parameters">

1. `Name`: Record name.
2. `Model`: Model name.
3. `Dataset`: Row that identifies the dataset the model was trained on.
4. `Result`: Values of metrics on the dataset and/or its separate slices.
5. `Tags`: Set of tags used for filtering results.
6. `ID`: Identifier.
7. `Username`: User who created the run or experiment.
8. `Datetime`: Run date.
9. `ModelInfo`: Model properties (a set of parameters in JSON format).
10. `ModelVersion`: Model version.
11. `DatasetInfo`: Dataset properties.
12. `Status`: Status.
13. `Description`: description.

Runs and experiments also contain:

* [Results per object](#slices).

</Cut>
