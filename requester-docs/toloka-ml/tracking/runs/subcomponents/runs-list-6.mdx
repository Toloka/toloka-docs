Open the Toloka ML platform [Runs](https://tolokamodels.tech/runs) page.

To view runs by multiple parameters, edit the appropriate fields in the search menu at the top:

* **Model names**: Model name.
* **Dataset names**: Dataset name.
* **Username**: Run owner's username.
* **Tags**: Set of run tags.
* **Date from**: Start date to view runs from.
* **Date to**: End date to view runs to.
* **All**: All runs with any status.
    * Running: Graph is running.
    * Completed: Graph completed successfully. If the run was added from the Python library, it will always have this status.
    * Failed: Graph failed or wasn't run due to an error.
