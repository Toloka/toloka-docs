To add a run to Toloka ML platform, set the `ModelInfo`, `Metrics`, `Metric`, and `DatasetInfo` objects. Next, create metrics and a run object and pass your [OAuth token](/docs/toloka-ml/api/#token) and the run object to the client.

```py
# The metrics object
metrics = Metrics(
    [
        Metric('AUC', 0.85, 'slice1', tags=['tag1', 'tag2'], main=True),
        Metric('Accuracy', 0.93, 'slice1', tags=['tag2'], main=True),
        Metric('Calc', [0.88, 0,92], tags=['tag3', 'tag2']),
    ]
)

run = RunInfo(
    # The ModelInfo object
    models=models,
    # The DatasetInfo object
    datasets=datasets,
    # The metrics object
    metrics=metrics,

    # (Optional) The list of tags
    tags=['test_tag_44'],

    # (Optional) The user date and time
    user_datetime=dt(2019, 8, 11),

    # (Optional) The run name
    name='Some_name',

    # (Optional) The run description
    description='Some description.',

    # (Optional) The link to the object data
    model_output_url=yt_path,
)

client = PulsarClient(token='your_token')

run_id = client.add(run)
print(run_id)
```
