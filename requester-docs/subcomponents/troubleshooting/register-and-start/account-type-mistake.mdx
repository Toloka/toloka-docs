<Cut title="I became a requester by mistake, how can I use the platform as a Toloker and earn money?">

If you want to complete tasks and earn money, you need to register in Toloka once again, with a different username, but as a Toloker.

To create a Toloker account:

1. Log out of your requester account.

1. Go to the [Toloka](https://toloka.ai) home page.

1. Click **Join**.

1. Follow the system instructions.

<Note mode="note" title="Note">

For a step-by-step description of the registration process, see [Registration and login](https://join.toloka.ai/en/docs/web/register/#registered).

</Note>

</Cut>
