To make sure that the Toloker listened to the entire recording, use the `condition.played-fully` component in the `validation` property.

```json
{
  "type": "view.audio",
  "url": "http://example.com/audio.mp3",
  "validation": {
    "type": "condition.played-fully"
  }
}
```
