You can set up shortcuts to let Tolokers confirm or cancel area creation. Assign shortcuts to properties:

  - `confirm`: Confirm.
  - `cancel`: Cancel.

```json
{
    "type": "plugin.field.image-annotation.hotkeys",
    "confirm": "x",
    "cancel": "z"
}
```
