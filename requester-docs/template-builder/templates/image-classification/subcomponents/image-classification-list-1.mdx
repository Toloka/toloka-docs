If there are several possible answers to the question, add a group of checkboxes using the [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group) component.

<Cut title="Show code">

```json
{
  "type": "field.checkbox-group",
  "label": "What do you see in the picture?",
  "options": [
    {
      "label": "Crosswalk",
      "value": "line"
    },
    {
      "label": "Cars",
      "value": "cars"
    },
    {
      "label": "Pedestrians",
      "value": "walkers"
        },
    {
      "label": "Parking",
      "value": "parking"
    },
    {
      "label": "Road signs",
      "value": "road-signs"
    }
  ],
  "validation": {
    "type": "condition.required",
    "hint": "select at least one answer"
  },
  "data": {
    "type": "data.output",
    "path": "result"
  }
}
```

</Cut>
