# Named Entity Recognition (NER)

For this type of project, you can use the **Named Entity Recognition (NER)** preset.

This preset helps you identify parts of speech, proper nouns, and other entities in the text.

Take a look at the example: the labeling interface includes a text block and buttons for categories.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/-WB38I-Y3xvU3M">View example</Button>

<Cut title="Components used in the example">

- [field.text-annotation](/docs/template-builder/reference/field.text-annotation): A component for text segmentation.

    <Cut title="Show code">

    ```json
    {
      "type": "field.text-annotation",
      "content": {
        "type": "data.input",
        "path": "input"
      }
    }
    ```

    </Cut>

- [condition.required](/docs/template-builder/reference/condition.required): Checks that the data is filled in.

    <Cut title="Show code">

    ```json
    {
      "type": "condition.required"
    }
    ```

    </Cut>

</Cut>

## Add a description {#add-description}

To add a detailed description to the task, use the [view.text](/docs/template-builder/reference/view.text) component.

<Cut title="Show code">

```json
{
  "type": "view.text",
  "content": "In the text provided, find information about the product and manufacturer and highlight it with different colors: red for product name, blue for brand, green for package size, pink for country of manufacture. If a word is misspelled, highlight it with yellow."
}
```

</Cut>

## Add a response field {#add-text-area}

To let Tolokers leave comments about the task or their response, add a text field using [field.textarea](/docs/template-builder/reference/field.textarea).

<Cut title="Show code">

```json
{
  "type": "field.textarea",
  "label": "Comments",
  "placeholder": "Enter text",
  "data": {
    "type": "data.output",
    "path": "comment"
  }
}
```

</Cut>

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/FYqX4xd63yQyPz">View example</Button>

## Add a layout {#add-layout}

To enhance Toloker's experience, you can highlight different types of data with colors using [view.alert](/docs/template-builder/reference/view.alert). In this example, the text is highlighted with a blue border.

<Cut title="Show code">

```json
{
  "type": "view.alert",
  "theme": "info",
  "content": {
    "type": "view.text",
    "content": "In the text provided, find information about the product and manufacturer and highlight it with different colors: red for product name, blue for brand, green for package size, pink for country of manufacture. If a word is misspelled, highlight it with yellow."
  }
}
  ```

</Cut>

## Add keyboard shortcuts {#add-shortcuts}

Assign keyboard shortcuts for selecting categories using [plugin.field.text-annotation.hotkeys](/docs/template-builder/reference/plugin.field.text-annotation.hotkeys) configuration.

<Cut title="Show code">

```json
"plugins": [
  {
    "type": "plugin.field.text-annotation.hotkeys",
    "labels": [
      "1",
      "2",
      "3",
      "4",
      "5"
    ]
  }
]
```

</Cut>
