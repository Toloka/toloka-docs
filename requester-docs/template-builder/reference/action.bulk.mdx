# action.bulk

Use this component to call multiple [actions](/docs/template-builder/reference/actions) at the same time, like to show more than one notification when a button is clicked.

Actions are invoked in the order in which they are listed. This means that if two actions write a value to the same variable, the variable will always have the second value.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/jH6BvDpE3twfMn">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [view.text](/docs/template-builder/reference/view.text): Displays a text.
- [view.action-button](/docs/template-builder/reference/view.action-button): Displays a button that calls an action.
- [action.notify](/docs/template-builder/reference/action.notify): Adds a component that creates a message in the lower-left corner of the screen.
