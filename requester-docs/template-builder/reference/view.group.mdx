# view.group

Groups components visually into framed blocks.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/eYAQsjHW48x8ZG">View example</Button>

<Cut title="Components used in the example">

- [layout.columns](/docs/template-builder/reference/layout.columns): Places content in columns.
- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [field.radio-group](/docs/template-builder/reference/field.radio-group): Adds radio buttons for selecting an answer option.
- [data.output](/docs/template-builder/operations/work-with-data): The output data. This is what you get when a Toloker clicks the **Submit** button.
- [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group): Adds a group of checkboxes for selecting independent answer options.
- [view.alert](/docs/template-builder/reference/view.alert): Creates a color block to highlight important information.
- [view.text](/docs/template-builder/reference/view.text): Displays a text.

</Cut>

## Component properties {#properties}

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style="color: red">\*</span> | "view.group" | Set component type.
`label` | _string_ | Group heading.
`content` | _view_ | Content of a group block.
`hint` | _string_ | Explanation of the group heading. To insert a new line, use `\n`.
`validation` | _condition_ | Validation based on condition.
