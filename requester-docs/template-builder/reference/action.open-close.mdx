# action.open-close

This component changes the display mode of another component by opening or closing it. What happens to the component depends on the type of component:

- [view.image](/docs/template-builder/reference/view.image) — expands the image to full screen.
- [view.collapse](/docs/template-builder/reference/view.collapse) — expands or collapses a collapsible section of content.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/5rw-rEtC3tzAcs">View example</Button>

<Cut title="Components used in the example">

- [layout.sidebar](/docs/template-builder/reference/layout.sidebar): Lets you place the main content block and an adjacent panel with controls on a page.
- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [view.collapse](/docs/template-builder/reference/view.collapse): Displays an expandable block.
- [view.text](/docs/template-builder/reference/view.text): Displays a text.
- [view.image](/docs/template-builder/reference/view.image): Displays an image.
- [view.action-button](/docs/template-builder/reference/view.action-button): Displays a button that calls an action.

</Cut>
