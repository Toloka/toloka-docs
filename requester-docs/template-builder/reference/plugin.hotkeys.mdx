# plugin.hotkeys

You can use this to set keyboard shortcuts for actions.

[Learn more about hotkeys](/docs/template-builder/best-practices/hotkeys).

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/0XptBuBn4QhWYJ">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group): Adds a group of checkboxes for selecting independent answer options.
- [data.output](/docs/template-builder/operations/work-with-data): The output data. This is what you get when a Toloker clicks the **Submit** button.
- [condition.required](/docs/template-builder/reference/condition.required): Checks if an option is selected.
- [action.toggle](/docs/template-builder/reference/action.toggle): Changes the value in the data from `true` to `false` and vice versa. 

</Cut>

## Component properties `{#properties}`

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style={{color: "red"}}>\*</span> | "plugin.hotkeys" | Set component type.
`a-z\|0-9\|up\|down` | _string_ | Select a key (a Latin letter, a number, or an up/down arrow) and configure it to do the same action as the component that you assign it to. The action is called when the selected key is pressed.
