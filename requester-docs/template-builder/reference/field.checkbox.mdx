# field.checkbox

Adds a single checkbox.

If you need to add a group of checkboxes, use [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group).

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/xRm39MZ-47HsVq">View example</Button>

<Cut title="Components used in the example">

- [data.output](/docs/template-builder/operations/work-with-data): The output data. This is what you get when a Toloker clicks the **Submit** button.
- [condition.required](/docs/template-builder/reference/condition.required): Checks if an option is selected.

</Cut>

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style="color: red">\*</span> | "field.checkbox" | Set component type.
