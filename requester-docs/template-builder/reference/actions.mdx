# List of actions

Use the components named `actions.*` to perform various actions, such as [showing notifications](/docs/template-builder/reference/action.notify).

| Component                                 | Description                                                                                                |
| ----------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| [action.bulk](/docs/template-builder/reference/action.bulk)             | Use this component to call multiple actions at the same time.                                              |
| [action.notify](/docs/template-builder/reference/action.notify)         | The component creates a message in the lower-left corner of the screen.                                    |
| [action.open-close](/docs/template-builder/reference/action.open-close) | Changes the display mode for another component — opens or closes it.                                       |
| [action.open-link](/docs/template-builder/reference/action.open-link)   | Opens a new tab in the browser with the specified web page.                                                |
| [action.play-pause](/docs/template-builder/reference/action.play-pause) | This component controls audio or video playback. It stops playback in progress or starts if it is stopped. |
| [action.rotate](/docs/template-builder/reference/action.rotate)         | Rotates the specified component by 90 degrees.                                                             |
| [action.set](/docs/template-builder/reference/action.set)               | Sets the value from `payload` in the data in the `data` property.                                          |
| [action.toggle](/docs/template-builder/reference/action.toggle)         | The component changes the value in the data from `true` to `false` and vice versa.                         |
