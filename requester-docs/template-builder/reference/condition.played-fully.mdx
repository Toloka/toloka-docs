# condition.played-fully

This component checks for the end of playback. Validation is passed if playback is finished. To play media with the `condition.played-fully` check, you can use [view.audio](/docs/template-builder/reference/view.audio) and [view.video](/docs/template-builder/reference/view.video). The `condition.played-fully` check only works in the player's `validation` property.
