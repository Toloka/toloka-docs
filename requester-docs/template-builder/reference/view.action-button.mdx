# view.action-button

Button that calls an action.

When clicking the button, an action specified in the `action` property is called.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/BiV4rmEt45Vu5w">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [view.text](/docs/template-builder/reference/view.text): Adds a block with text.
- [action.open-link](/docs/template-builder/reference/action.open-link): Opens a new tab in the browser with the specified web-page.

</Cut>

## Component properties {#properties}

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style="color: red">\*</span> | "view.action-button" | Set component type.
`label` | _string_ | Button text.
`action` | _action_ | Action called when clicking the button.
`hint` | _string_ | Hint text.
`validation` | _condition_ | Validation based on condition.
