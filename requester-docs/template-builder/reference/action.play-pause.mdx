# action.play-pause

This component controls audio or video playback. It stops playback in progress or starts if it is stopped.

For example, this component will allow you to play two videos simultaneously.

<Button view="defaultInverse" size="l" type="link" url="https://ya.cc/t/AiIEBqw-3YbMBH">View example</Button>

<Cut title="Components used in the example">

- [layout.side-by-side](/docs/template-builder/reference/layout.side-by-side): Displays several data blocks of the same width on a single horizontal panel.
- [view.video](/docs/template-builder/reference/view.text): Adds a player for video playback.
- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [field.radio-group](/docs/template-builder/reference/field.radio-group): Adds a component for selecting one value out of several options.
- [condition.required](/docs/template-builder/reference/condition.required): Checks that the data is filled in.
- [view.action-button](/docs/template-builder/reference/view.action-button): Displays a button that calls an action.
- [action.bulk](/docs/template-builder/reference/action.bulk): Calls multiple [actions](/docs/template-builder/reference/actions) at the same time.

</Cut>

You can also stop or start playback for some event ([plugin.trigger](/docs/template-builder/reference/plugin.trigger)) or by pressing the hotkey ([plugin.hotkeys](/docs/template-builder/reference/plugin.hotkeys)).
