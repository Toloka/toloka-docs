# view.divider

The component displays a horizontal delimiter.

You can place extra elements in the center of the delimiter, like a popup `hint` and `label`.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/yWUusmRj48ZRya">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [view.image](/docs/template-builder/reference/view.image): Displays an image.
- [data.input](/docs/template-builder/operations/work-with-data): The input data. For example, links to images that will be shown to Tolokers.
- [view.text](/docs/template-builder/reference/view.text): Displays a text.
- [field.button-radio-group](/docs/template-builder/reference/field.button-radio-group): Adds a group of buttons for selecting an answer option.
- [condition.required](/docs/template-builder/reference/condition.required): Checks that the data is filled in.
- [data.output](/docs/template-builder/operations/work-with-data): The output data. This is what you get when a Toloker clicks the **Submit** button.

</Cut>

## Component properties {#properties}

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style="color: red">\*</span> | "view.divider" | Set component type.
`label` | _string_ | A label in the center of the delimiter. Line breaks are not supported.
`hint` | _string_ | Hint text.
`validation` | _condition_ | Validation based on condition.
