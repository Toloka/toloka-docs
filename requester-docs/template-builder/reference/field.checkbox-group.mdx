# field.checkbox-group

Adds a group of checkboxes for selecting independent answer options. This is useful if there are several options. You can add just one checkbox, but it's easier to use [field.checkbox](/docs/template-builder/reference/field.checkbox) for that.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/qeQoTGCe47BNW2">View example</Button>

<Cut title="Components used in the example">

- [data.output](/docs/template-builder/operations/work-with-data): The output data. This is what you get when a Toloker clicks the **Submit** button.
- [condition.required](/docs/template-builder/reference/condition.required): Checks if at least one option is selected.

</Cut>

By default, if the component returns `false`, this result will not be added to the output. To add `false` to the results, specify `"preserveFalse": true`.
