# action.toggle

The component changes the value in the data from `true` to `false` and vice versa.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/bJN7DmZV4GAR8K">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group): Adds a group of checkboxes for selecting independent answer options.
- [plugin.hotkeys](/docs/template-builder/reference/plugin.hotkeys): Sets keyboard shortcuts for actions.

</Cut>

## Component properties `{#properties}`

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style={{color: "red"}}>\*</span> | "action.toggle" | Set component type.
`data` | _writable_ | Data in which the value will be changed. The data type must be `boolean`.
