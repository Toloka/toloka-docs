# view.labeled-list

Displaying components as a list with labels placed on the left.

<Button view="defaultInverse" size="l" type="link" url="https://ya.cc/t/UurufJ-t4JLFCP">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [view.text](/docs/template-builder/reference/view.text): Adds a block with text.
- [view.link](/docs/template-builder/reference/view.link): Displays a link.
- [field.button-radio-group](/docs/template-builder/reference/field.button-radio-group): Adds buttons for selecting an answer option.

</Cut>

If you don't need labels, use [view.list](/docs/template-builder/reference/view.list).

## Component properties `{#properties}`

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style={{color: "red"}}>\*</span> | "view.labeled-list" | Set component type.
`label` | _string_ | Label above the component.
`hint` | _string_ | Hint text.
`items`<span style={{color: "red"}}>\*</span> | _array of objects_ | An array of list items.
`items[]` | _object_ | Parameters of a list item.
`items[].centerLabel` | _boolean_ | If `true`, a `label` is center-aligned relative to the content of a list item (`content`). Use it if the list consists of large items, such as images or multi-line text.<br/><br/>By default, `false` (the label is aligned to the top of the content block).
`items[].content`<span style={{color: "red"}}>\*</span> | _view_ | List item content.
`items[].hint` | _string_ | A pop-up hint displayed next to a label.
`items[].label`<span style={{color: "red"}}>\*</span> | _string_ | A label displayed next to a list item.
`minWidth` | _number_ | The minimum width of list content. If the component width is less than the specified value, it switches to compact mode.
`validation` | _condition_ | Validation based on condition.
