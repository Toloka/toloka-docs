# layout.side-by-side

The component displays several data blocks of the same width on a single horizontal panel. For example, you can use this to compare several photos.

You can add selection buttons or a field for comment under the compared elements.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/E15DUzkl44n8nE">View example</Button>

<Cut title="Components used in the example">

- [action.set](/docs/template-builder/reference/action.set): Sets the value from `payload` in the data in the `data` property.
- [plugin.hotkeys](/docs/template-builder/reference/plugin.hotkeys): Sets keyboard shortcuts for actions.
- [layout.side-by-side](/docs/template-builder/reference/layout.side-by-side): Displays several data blocks of the same width on a single horizontal panel.
- [view.image](/docs/template-builder/reference/view.image): Adds an image.
- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [field.radio-group](/docs/template-builder/reference/field.radio-group): Adds a group of radio buttons for selecting an answer option.
- [field.textarea](/docs/template-builder/reference/field.textarea): Adds a text block to enter milti-line text.
- [condition.required](/docs/template-builder/reference/condition.required): Checks that the data is filled in.

</Cut>

Differences from the [layout.compare](/docs/template-builder/reference/layout.compare) component:

- You can add buttons which allow hiding the elements. Use it if you need to compare, for example, 5 photos at once and need to choose between two.
- You can configure only common control elements. You can't add separate buttons under each of the compared element.

## Component properties {#properties}

**Name** | **Type** | **Description**
----- | ----- | -----
`type`<span style="color: red">\*</span> | "layout.side-by-side" | Set component type.
`controls`<span style="color: red">\*</span> | _view_ | Components that let Tolokers perform the required actions.<br/><br/>For example: [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group) or [field.button-radio-group](/docs/template-builder/reference/field.button-radio-group).
`items`<span style="color: red">\*</span> | _array_ | An array of data blocks.
`items[]` | _view_ | A component for data output or input. For example, [view.image](/docs/template-builder/reference/view.image).
`minItemWidth` | _number_ | The minimum width of a data block, at least 400 pixels.
`validation` | _condition_ | Validation based on condition.
