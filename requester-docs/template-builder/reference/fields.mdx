# List of data entry elements

Use the components named `field.*` as the data entry fields, such as a [text box](/docs/template-builder/reference/field.text) or [dropdown list](/docs/template-builder/reference/field.select). Be sure to specify where to save the data from such elements.

| Component                                               | Description                                                                                                                                                                                                           |
| ------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [field.audio](/docs/template-builder/reference/field.audio)                           | Component for recording audio. Works in the [Toloka app for mobile devices](https://toloka.ai/tolokers/docs/mobile/?lang=en). In a browser, this component opens a window for uploading an audio file.                |
| [field.button-radio](/docs/template-builder/reference/field.button-radio)             | Adds a button. This is useful for creating an optional answer.                                                                                                                                                        |
| [field.button-radio-group](/docs/template-builder/reference/field.button-radio-group) | Adds buttons for selecting an answer option. This is useful if there are several options. You can add just one button, but it's easier to use [field.button-radio](/docs/template-builder/reference/field.button-radio) for that.                   |
| [field.checkbox](/docs/template-builder/reference/field.checkbox)                     | A checkbox.                                                                                                                                                                                                           |
| [field.checkbox-group](/docs/template-builder/reference/field.checkbox-group)         | Adds a group of checkboxes for selecting independent answer options. This is useful if there are several options. You can add just one checkbox, but it's easier to use [field.checkbox](/docs/template-builder/reference/field.checkbox) for that. |
| [field.date](/docs/template-builder/reference/field.date)                             | A component for entering the date and time in the desired format and range.                                                                                                                                           |
| [field.email](/docs/template-builder/reference/field.email)                           | Creates a field for entering an email address.                                                                                                                                                                        |
| [field.file](/docs/template-builder/reference/field.file)                             | This component can be used for uploading files. It's displayed in the interface as an upload button.                                                                                                                  |
| [field.image-annotation](/docs/template-builder/reference/field.image-annotation)     | Component for image labeling.                                                                                                                                                                                         |
| [field.list](/docs/template-builder/reference/field.list)                             | A component that allows a Toloker to add and remove list items, such as text fields to fill in. This way you can allow a Toloker to give multiple answers to a question.                                              |
| [field.map](/docs/template-builder/reference/field.map)                               | Adds a map to your task and allows Tolokers to set markers on it.                                                                                                                                                     |
| [field.media-file](/docs/template-builder/reference/field.media-file)                 | Adds buttons for different types of uploads: uploading photos or videos, selecting files from the file manager or choosing from the gallery.                                                                          |
| [field.number](/docs/template-builder/reference/field.number)                         | A component that allows a Toloker to enter a number.                                                                                                                                                                  |
| [field.phone-number](/docs/template-builder/reference/field.phone-number)             | Creates a field for entering a phone number.                                                                                                                                                                          |
| [field.quiz](/docs/template-builder/reference/field.quiz)                             | A quiz component that counts only the first answered values.                                                                                                                                                                 |
| [field.radio-group](/docs/template-builder/reference/field.radio-group)               | A component for selecting one value out of several options. It is designed as a group of circles arranged vertically.                                                                                                 |
| [field.select](/docs/template-builder/reference/field.select)                         | Button for selecting from a drop-down list. Use this component when the list is long and only one option can be chosen.                                                                                               |
| [field.text](/docs/template-builder/reference/field.text)                             | A field for entering a short text, not more than a line. To add a field for entering multiple lines of text, use [field.textarea](/docs/template-builder/reference/field.textarea).                                                                 |
| [field.text-annotation](/docs/template-builder/reference/field.text-annotation)       | A component for text segmentation.                                                                                                                                                                                    |
| [field.textarea](/docs/template-builder/reference/field.textarea)                     | Box for entering multi-line text.                                                                                                                                                                                     |
