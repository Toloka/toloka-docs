# view.audio

The component plays audio.

Format support depends on the Toloker's browser, OS, and device. We recommend using MP3.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/d0D-SzfG46oojL">View example</Button>

<Cut title="Components used in the example">

- [view.list](/docs/template-builder/reference/view.list): Displays data in a list.
- [data.input](/docs/template-builder/operations/work-with-data): The input data. For example, a link to an audio recording that Toloker will listen to.
- [data.output](/docs/template-builder/operations/work-with-data): The output data. This is what you get when a Toloker clicks the **Submit** button.
- [condition.played-fully](/docs/template-builder/reference/condition.played): Checks if a Toloker listened to the entire audio track.
- [field.button-radio-group](/docs/template-builder/reference/field.button-radio-group): Adds a group of buttons for selecting an answer option.
- [condition.required](/docs/template-builder/reference/condition.required): Checks if at least one option is selected.

</Cut>
