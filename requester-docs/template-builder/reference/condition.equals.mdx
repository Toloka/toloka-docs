# condition.equals

Checks whether the original value is equal to the specified value. If it matches, it returns `true`, otherwise it returns `false`.

When substituting values, you can refer to `data.*` or another element using `$ref`. You can also use [helpers](/docs/template-builder/reference/helpers) and [conditions](/docs/template-builder/reference/conditions) to get the value.
