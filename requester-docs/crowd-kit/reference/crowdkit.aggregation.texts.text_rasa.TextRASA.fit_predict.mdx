| Parameters | Type | Description |

`true_objects`|**Optional\[Series\]**|<p>Tasks' ground truth texts. A pandas.Series indexed by `task` such that `labels.loc[task]` is the tasks's ground truth text.</p>

* **Returns:**

  Tasks' texts.
