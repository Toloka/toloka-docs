# get_most_probable_labels

`crowdkit.aggregation.utils.get_most_probable_labels` | [Source code](https://github.com/Toloka/crowd-kit/blob/v1.2.1/crowdkit/aggregation/utils.py#L73)

```python
get_most_probable_labels(proba: DataFrame)
```

Returns most probable labels

## Parameters description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`proba`|**DataFrame**|<p>Tasks' label probability distributions. A pandas.DataFrame indexed by `task` such that `result.loc[task, label]` is the probability of `task`'s true label to be equal to `label`. Each probability is between 0 and 1, all task's probabilities should sum up to 1</p>
