# Overview

A skill is an assessment of some aspect of a Toloker's responses (a number from 0 to 100). You can set up skill calculation in a [quality control rule](/docs/api/quality_control), or manually set the skill level for a Toloker. You can use skills to [select Tolokers](/docs/api/filter-skill) who perform pool tasks.

## Methods {#methods}

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/api-reference/#tag--skill):** more parameter details, request/response examples, and code samples in various programming languages, including the [Toloka-Kit](/docs/toloka-kit/) usage samples.

</Note>

Method | Endpoint | Overview
----- | ----- | -----
POST | [/skills](/docs/api/create-skill) | Creates a skill.
PUT | [/skills/<id\>](/docs/api/edit-skill) | Changes the name, comment, and access to the skill.
PUT | [/user-skills](/docs/api/set-skill) | Sets the skill value.
GET | [/user-skills](/docs/api/get-user-skill-list) | Gets а list of Tolokers who have skills.
GET | [/user-skills/<id\>](/docs/api/get-user-skill) | Gets a Toloker's skill value.
GET | [/skills](/docs/api/get-skill-list) | Gets the list of created skills.
GET | [/skills/<id\>](/docs/api/get-skill) | Gets the properties of a skill.
DELETE | [/user-skills/<id\>](/docs/api/delete-skill) | Removes a skill for a Toloker.
