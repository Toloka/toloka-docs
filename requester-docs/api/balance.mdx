## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Response {#response}

Contains information about the requester and the account balance:

```json
{
  "balance": 121.30,
  "id": "566ec2b0ff0deeaae5f9d500",
  "public_name": {
    "EN": "Ya.Shield"
  }
}
```

Parameter | Overview
----- | -----
**balance** | **float**<br/><br/>Account balance in US dollars.
**id** | **string**<br/><br/>Requester ID.
**public_name** | **object**<br/><br/>The requester's name in Toloka.
