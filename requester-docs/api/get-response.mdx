# Overview

You can get:

- Responses for the pool's task suites and their properties.
- [Files uploaded by Tolokers](/docs/api/attachments).
- [Aggregated responses](/docs/api/aggregated-solutions).

## Methods {#methods}

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/api-reference/#tag--assignment):** more parameter details, request/response examples, and code samples in various programming languages, including the [Toloka-Kit](/docs/toloka-kit/) usage samples.

</Note>

Method | Endpoint | Overview
----- | ----- | -----
GET | [/assignments](/docs/api/result) | Gets responses for all the pool's task suites.
GET | [/assignments/&lt;id&gt;](/docs/api/get-assignment-id) | Gets the properties of this response.
