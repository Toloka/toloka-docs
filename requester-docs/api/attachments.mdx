# Overview

Files uploaded by Tolokers are saved in Toloka.

## Methods {#methods}

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/api-reference/#tag--attachment):** more parameter details, request/response examples, and code samples in various programming languages, including the [Toloka-Kit](/docs/toloka-kit/) usage samples.

</Note>

Method | Endpoint | Overview
----- | ----- | -----
GET | [/attachments](/docs/api/get-attachment-list) | Gets the list of files saved in Toloka.
GET | [/attachments/<file_id>](/docs/api/get-attachment) | Gets the properties of a file attached to a task response.
GET | [/attachments/<file_id>/download](/docs/api/download-attachment) | Downloads a file attached to a task response.
