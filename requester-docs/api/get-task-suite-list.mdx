# Get the list of task suites

<Announce/>

Gets the list of task suites in the pool.

## Request {#request}

<Tabs tabs={[
  { title: 'Production version', value: <GetTaskSuiteListList1/> },
  { title: 'Sandbox', value: <GetTaskSuiteListList2/> },
]}/>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Query parameters {#query-params}

<QueryParams/>

Parameter | Overview
----- | -----
**pool_id** | **string**<br/><br/>ID of the pool to get task suites from.<br/><br/>You must specify either `task_id` or `pool_id`.
**task_id** | **string**<br/><br/>The task ID in the suites that were generated automatically using "smart mixing". You will get task suites that contain the specified task.<br/><br/>You must specify either `task_id` or `pool_id`.
**sort** | <GetTaskSuiteListTable1/>
**overlap** | **integer**<br/><br/>Suites with an overlap equal to the specified value.
**Standard query parameters** | [limit](/docs/api/standard-query-parameters#limit), [id_gt](/docs/api/standard-query-parameters#id_gt), [id_gte](/docs/api/standard-query-parameters#id_gte), [id_lt](/docs/api/standard-query-parameters#id_lt), [id_lte](/docs/api/standard-query-parameters#id_lte), [created_gt](/docs/api/standard-query-parameters#created_gt), [created_gte](/docs/api/standard-query-parameters#created_gte), [created_lt](/docs/api/standard-query-parameters#created_lt), [created_lte](/docs/api/standard-query-parameters#created_lte), [overlap_gt](/docs/api/standard-query-parameters#overlap_gt), [overlap_gte](/docs/api/standard-query-parameters#overlap_gte) [overlap_lt](/docs/api/standard-query-parameters#overlap_lt), [overlap_lte](/docs/api/standard-query-parameters#overlap_lte).

## Query example {#request-example}

You can set up the display of the list of task suites in parts (for example, 10 task suites at a time):

1. Show the first 10 task suites, starting from the one with the lowest ID.
1. Show the remaining task suites (10 at a time) in ascending order.
