```json
{
  "items": {
    "0": {details of a bonus #0},
    "2": {details of a bonus #2},
    "<N>": {details of a bonus #N}
  },
  "validation_errors": {
    "1": {validation errors for a bonus #1},
    "3": {validation errors for a bonus #3},
    "<N>": {validation errors for a bonus #N}
  }
}
```

Parameter | Overview
----- | -----
**items** | **string**<br/><br/>    Object with information about bonuses issued.
**validation_errors** | **string**<br/><br/>    An object with validation errors. Returned if the request has the parameter `skip_invalid_items=true`.
