# Toloka REST API

Manage projects using Toloka API. Integrate Toloka methods into your own applications and automate your interaction with the Toloka platform.

## [Toloka API](/docs/api/)

Get access to projects with the help of Toloka API. Manage projects, pool, upload tasks, and get the results.

- [Accessing the API](/docs/api/access)
- [Quick start](/docs/api/quickstart)
- [Projects](/docs/api/project)

Apps API is used to power Toloka bespoke solutions. It has all the means to run your project and get the labeling results programmatically.
