**boolean**

Shows whether the list is complete.

Acceptable values:

- `true` — Not all elements are included in the output due to restrictions in the [limit](/docs/api/standard-query-parameters#limit) parameter.

    You can get the rest of the results by using the `_lt(e)` and `_gt(e)` parameters.

    Learn more about [standard query parameters](/docs/api/standard-query-parameters).

- `false` — The output contains all the items.
