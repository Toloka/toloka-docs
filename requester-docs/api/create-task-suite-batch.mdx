# Create multiple task suites

<Announce/>

Creates multiple task suites.

<Note mode="restriction" title="Restriction">

You can add a maximum of 100,000 tasks per minute and a maximum of 2,000,000 tasks per day. Refer to the [Rate limiting](/docs/api/rate-limiting) section for the complete list of the request limitations in Toloka API.

</Note>
