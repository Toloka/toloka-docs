_string_<br/><br/>The languages the Toloker entered in the profile (uppercase two-letter ISO code conforming to [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes)).

Value of `operator`:

- `IN` — Contained in the list of regions or languages in the profile.
- `NOT_IN` — Not contained in the list of regions or languages in the profile.
