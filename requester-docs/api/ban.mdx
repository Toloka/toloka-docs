# Overview

You can block a Toloker's access to one or more projects. This lets you control which Tolokers will complete tasks. For example, you can select Tolokers with a skill value lower than N and block their access to tasks. You can also unblock access.

## Methods {#methods}

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/api-reference/#tag--user-restriction):** more parameter details, request/response examples, and code samples in various programming languages, including the [Toloka-Kit](/docs/toloka-kit/) usage samples.

</Note>

Method | Endpoint | Overview
----- | ----- | -----
PUT | [/user-restrictions](/docs/api/ban-create) | Blocks a Toloker from accessing tasks.
GET | [/user-restrictions](/docs/api/ban-get-list) | Gets the list of Toloker bans.
GET | [/user-restrictions/<ban_id>](/docs/api/ban-get-info) | Gets information about a ban.
DELETE | [/user-restrictions/<ban_id>](/docs/api/ban-delete) | Removes a ban.
