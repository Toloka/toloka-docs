# Overview

You can issue rewards to one or more Tolokers (in addition to payment for completed tasks).

The reward amount can be from $0.005 to $100 per Toloker per time.

## Methods {#methods}

Method | Endpoint | Overview
----- | ----- | -----
POST | [/user-bonuses](/docs/api/create-bonus) | Issues rewards to Tolokers.
GET | [/user-bonuses](/docs/api/get-bonus-list) | Gets a list of rewards issued.
GET | [/user-bonuses/<bonus_id>](/docs/api/get-one-bonus) | Gets the properties of the reward issued.
