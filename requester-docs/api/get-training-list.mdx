# Get a list of training pools

<Announce/>

Gets the list of created training pools (including archived pools).

## Request {#request}

<Tabs tabs={[
  { title: 'Production version', value: <GetTrainingListList1/> },
  { title: 'Sandbox', value: <GetTrainingListList2/> },
]}/>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Query parameters {#query-params}

Specified in the link after the question mark; separated by `&`.

Parameter | Overview
----- | -----
**status** | <GetTrainingListTable1/>
**project_id** | **string**<br/><br/>ID of the project that the training pool is linked to.
**sort** | <GetTrainingListTable2/>
**Standard query parameters** | [limit](/docs/api/standard-query-parameters#limit), [id_gt](/docs/api/standard-query-parameters#id_gt), [id_gte](/docs/api/standard-query-parameters#id_gte), [id_lt](/docs/api/standard-query-parameters#id_lt), [id_lte](/docs/api/standard-query-parameters#id_lte), [created_gt](/docs/api/standard-query-parameters#created_gt), [created_gte](/docs/api/standard-query-parameters#created_gte), [created_lt](/docs/api/standard-query-parameters#created_lt), [created_lte](/docs/api/standard-query-parameters#created_lte), [last_started_gt](/docs/api/standard-query-parameters#last_started_gt), [last_started_gte](/docs/api/standard-query-parameters#last_started_gte), [last_started_gte](/docs/api/standard-query-parameters#last_started_gte), [last_started_lt](/docs/api/standard-query-parameters#last_started_lt), [last_started_lte](/docs/api/standard-query-parameters#last_started_lte).

## Query example {#request-example}

You can set up the display of the list of pools in parts (for example, 10 pools at a time):

1. Show the first 10 pools, starting with the pool with the lowest ID.
1. Show the remaining pools (10 at a time) in ascending order.

## Response {#response}

Contains a list of training pools and their properties in the `items` array:

```json
{"items" : [{parameters_of_training_pool_1}, {parameters_of_training_pool_2}, ... {parameters_of_training_pool__n_}], "has_more": false}
```
