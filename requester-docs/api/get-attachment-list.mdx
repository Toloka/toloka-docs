# Get the list of files

<Announce/>

Gets the list of files saved in Toloka.

<Note mode="warning" title="Warning">

Files are downloaded not in the order in which they were uploaded to the server.

</Note>

<Note mode="tip" title="Tip">

</Note>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Query parameters {#query-params}

<QueryParams/>

Parameter | Overview
----- | -----
**pool_id** | **string**<br/><br/>Pool ID.
**assignment_id** | **string**<br/><br/>ID of the task suite assignment.
**name** | **string**<br/><br/>File name.
**type** | **string**<br/><br/>Attachment type. Currently the key can have only one value — `ASSIGNMENT_ATTACHMENT`.
**user_id** | **string**<br/><br/>ID of the Toloker who uploaded the file(s).
**sort** | <GetAttachmentListTable1/>
**Standard query parameters** | [limit](/docs/api/standard-query-parameters#limit), [id_gt](/docs/api/standard-query-parameters#id_gt), [id_gte](/docs/api/standard-query-parameters#id_gte), [id_lt](/docs/api/standard-query-parameters#id_lt), [id_lte](/docs/api/standard-query-parameters#id_lte), [created_gt](/docs/api/standard-query-parameters#created_gt), [created_gte](/docs/api/standard-query-parameters#created_gte), [created_lt](/docs/api/standard-query-parameters#created_lt), [created_lte](/docs/api/standard-query-parameters#created_lte).

## Query example {#request-example}

You can set up the display of the list of files in parts (for example, 10 files at a time):

1. Show the first 10 files, starting with the one with the lowest ID.
1. Show the remaining files (10 at a time) in ascending order.
