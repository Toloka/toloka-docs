# Get operation details

<Announce/>

Gets operation details.

## Request {#query}

<Tabs tabs={[
  { title: 'Production version', value: <GetOperationList1/> },
  { title: 'Sandbox', value: <GetOperationList2/> },
]}/>

## Path parameters {#path-params}

Parameter | Overview
----- | -----
**operation_id** | Operation ID.

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Response {#response}

Contains operation details in JSON format (see the example in the [Operations](/docs/api/operations) section).

```json
{
  "id" : "57068577e4b0bf7b07a0256f",
  "type" : "TASK_SUITE.BATCH_CREATE",
  "status" : "FAIL",
  "submitted" : "2016-04-07T16:06:15.902",
  "started" : "2016-04-07T16:06:15.902",
  "finished" : "2016-04-07T16:06:15.902",
  "progress" : 100,
  "parameters" : {
    "open_pool" : fase,
    "allow_defaults" : false,
    "skip_invalid_items" : false
  },
  "details" : {
    "total_count" : 2,
    "valid_count" : 0,
    "failed_count" : 2,
    "success_count" : 0,
    "not_valid_count" : 2
  }
}
```

Parameter | Overview
----- | -----
**id** | **string \| required**<br/><br/>Operation ID.
**type** | <GetOperationTable1/>
**status** | <GetOperationTable2/>
**submitted** | **string \| required**<br/><br/>The UTC date and time the request was sent, in ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
**started** | **string**<br/><br/>The UTC date and time the operation started, in ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
**finished** | **string**<br/><br/>The UTC date and time the operation finished, in ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
**progress** | **integer**<br/><br/>The percentage of the operation completed.
**parameters** | **JSON**<br/><br/>Operation parameters (depending on the operation type).
**details** | **JSON**<br/><br/>Details of the operation completion.

The response doesn't provide a detailed error description. If an error has occurred, [request the operation log](/docs/api/get-operation-log).
