**boolean**<br/><br/>Overlap settings:

- `true` — Use the overlap that is set in the pool parameters (in the [defaults.default_overlap_for_new_task_suites](/docs/api/create-pool#default_overlap_for_new_task_suites) key).
- `false` — Use the overlap that is set in the task suite parameters (in the [overlap](/docs/api/create-task-suite#overlap) field).<br/><br/>The default value is `false`.
