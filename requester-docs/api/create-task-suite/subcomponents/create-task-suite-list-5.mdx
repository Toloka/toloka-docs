```json
{
  "items": {
    "0": {<task suite>},
    "2": {<task suite>}, ...
    "<n>": {<task suite N>}
  },
  "validation_errors": {
    "1": {<validation errors for the task suite>},
    "3": {<validation errors for the task suite>}, ...
    "<n>": {<validation errors for task suite N>}
  }
}
```

Parameter | Overview
----- | -----
**items** | **object**<br/><br/>An object with the created task suites.
**validation_errors** | **object**<br/><br/>An object with the errors in task suites. Returned if the request has the parameter `skip_invalid_items=true`.
**<n\>** | **object**<br/><br/>Sequential number of the task suite in the array (starting from 0).
