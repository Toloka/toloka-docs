**boolean**<br/><br/>Overlap settings:

- `true` — Use the overlap that is set in the pool parameters (in the [defaults.default_overlap_for_new_tasks](/docs/api/create-pool#default-overlap-tasks) key).
- `false` — Use the overlap that is set in the task parameters (in the [overlap](#overlap) field).<br/><br/>The default value is `false`.
