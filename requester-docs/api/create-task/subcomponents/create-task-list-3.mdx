Information about the created task. Besides [parameters](#body) that are set when creating a task, it includes parameters that are assigned to the task automatically:

Parameter | Overview
----- | -----
**id** | **string**<br/><br/>  Task ID.
**origin_task_id** | **string**<br/><br/>  The ID of a task from another pool from which this task was copied for majority vote verification.
**created** | **string**<br/><br/>  The UTC date and time when the task suite was created, in ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
