# Clone a training pool

<Announce/>

Creates a duplicate training pool.

An empty training pool will be created with the same parameters.

<Note mode="restriction" title="Restriction">

You can send a maximum of 20 requests of this kind per minute and 100 requests per day. Refer to the [Rate limiting](/docs/api/rate-limiting) section for the complete list of the request limitations in Toloka API.

</Note>
