# Overview

Send messages to Tolokers. You can specify recipients in a [list](/docs/api/message-send#spis) or select Tolokers using a [filter](/docs/api/message-send#didi).

The sent message is added to a new message thread. Until the first response is received, the message chain is located in the `UNREAD` folder. If the thread contains several addresses and one of them responds, a new message thread will be created.

## Methods {#methods}

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/api-reference/#tag--message-thread):** more parameter details, request/response examples, and code samples in various programming languages, including the [Toloka-Kit](/docs/toloka-kit/) usage samples.

Method | Endpoint | Overview
----- | ----- | -----
POST | [/message-threads/compose](/docs/api/message-send) | Sends a message to Tolokers.
POST | [/message-threads/<thread_id>/reply](/docs/api/reply-message) | Replies to a message thread.
GET | [/message-threads](/docs/api/get-chain-list) | Gets the list of message threads.
POST | [/message-threads/<thread_id>/add-to-folders](/docs/api/move-message) | Adds a message thread to a folder.
POST | [/message-threads/<thread_id>/remove-from-folders](/docs/api/move-message) | Deletes a message thread from a folder.
