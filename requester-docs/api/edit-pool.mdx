# Edit a pool

<Announce/>

Makes changes to a pool.

The pool can't be edited if it's open. [Close the pool](/docs/api/close-pool-for-update) before editing.

<Note mode="restriction" title="Restriction">

You can send a maximum of 10 requests per hour to change the pool overlap parameters values. Refer to the [Rate limiting](/docs/api/rate-limiting) section for the complete list of the request limitations in Toloka API.

</Note>

<Note mode="note" title="Note">

Learn about editing a training pool in the section [Edit a training pool](/docs/api/edit-training).

</Note>

Parameter | Overview

**pool_id** | Pool ID.

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.
**Content-Type** | Specifies the data format in the request body.

## Request body {#body}

In the request body, specify **all** the pool [parameters](/docs/api/create-pool#pool-param), including those that are updated.

## Response {#response}

Contains updated information about the pool (see the description in the [Create a pool](/docs/api/create-pool#response) section).
