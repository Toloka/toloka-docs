# Get a list of bans

<Announce/>

Gets the list of Toloker bans.

## Request {#request}

<Tabs tabs={[
  { title: 'Production version', value: <BanGetListList1/> },
  { title: 'Sandbox', value: <BanGetListList2/> },
]}/>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Query parameters {#query-params}

Specified in the link after the question mark; separated by `&`.

Parameter | Overview
----- | -----
**scope** | <BanGetListTable1/>
**user_id** | **string**<br/><br/>Toloker ID.
**project_id** | **string \| required if**<br/><br/>Required if `scope=PROJECT`.<br/><br/>The ID of the project that is blocked.
**pool_id** | **string \| required if**<br/><br/>Required if `scope=POOL`.<br/><br/>The ID of the pool that is blocked.
**sort** | <BanGetListTable2/>
**Standard query parameters** | [limit](/docs/api/standard-query-parameters#limit), [id_gt](/docs/api/standard-query-parameters#id_gt), [id_gte](/docs/api/standard-query-parameters#id_gte), [id_lt](/docs/api/standard-query-parameters#id_lt), [id_lte](/docs/api/standard-query-parameters#id_lte), [created_gt](/docs/api/standard-query-parameters#created_gt), [created_gte](/docs/api/standard-query-parameters#created_gte), [created_lt](/docs/api/standard-query-parameters#created_lt), [created_lte](/docs/api/standard-query-parameters#created_lte).

## Query example {#request-example}

To get the list of bans in parts (for example, 10 bans at a time), use a combination of parameters in requests:

<Tabs tabs={[
  { title: 'Production version', value: <BanGetListList3/> },
  { title: 'Sandbox', value: <BanGetListList4/> },
]}/>

## Response {#response}

Contains a list of bans in the `items` array:

```json
{"items" : [{ban parameters 1}, {ban parameters 2}, ... {ban parameters n}], "has_more": true}
```
