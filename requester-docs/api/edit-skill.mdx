Parameter | Overview

**id** | Skill ID.

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Request body {#body}

In the request body, specify **all** [the skill parameters](/docs/api/create-skill#body), including those that are updated.
