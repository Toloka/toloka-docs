# Регистрация и вход

<Note mode="warning" title="Важно">

Теперь в Толоке новый способ регистрации и авторизации. Если вы уже зарегистрированы в Толоке, но пока не обновляли данные для новой системы авторизации, то вам необходимо [сделать это](../new-authorization).

</Note>

## Регистрация `{#registered}`

Зарегистрироваться в Толоке можно с помощью аккаунта социальной сети:

1. Откройте [Толоку](https://toloka.ai/tolokers/ru/).

2. Нажмите кнопку **Начать**.

3. Нажмите кнопку соцсети, с помощью которой хотите зарегистрироваться. Чтобы увидеть все соцсети, нажмите ссылку **Другие способы**.

4. Откроется окно социальной сети. Выполните одно из действий:

    - Выберите аккаунт, если уже авторизованы.

    - Войдите в соцсеть, если не были авторизованы.

    - Создайте аккаунт, если пока не зарегистрированы в соцсети.

5. Введите номер телефона и нажмите кнопку **Отправить код**. Мы вышлем на этот номер SMS-сообщение с кодом подтверждения.

6. Введите код из SMS-сообщения и нажмите **Подтвердить**. Если не получили код или ввели некорректный, нажмите ссылку **Отправить код повторно**. Новый код можно запросить через минуту.

7. Заполните информацию о себе: имя и фамилия, дата рождения, страна. Ознакомьтесь с [Пользовательским соглашением](https://join.toloka.ai/legal/user-agreement-ru/), подтвердите, что вам исполнилось 18 лет, и нажмите кнопку **Продолжить**.

8. После успешной регистрации откроется [страница с заданиями](../task-select). Можно приступать к их выполнению.

## Вход `{#authorization}`

1. Откройте [Толоку](https://toloka.ai/tolokers/ru/).

2. Нажмите кнопку социальной сети.

3. В открывшемся окне социальной сети выполните одно из действий:

    - Выберите аккаунт, если уже авторизованы.

    - Войдите в соцсеть, если не были авторизованы.

4. Откроется [страница с заданиями](../task-select). Можно приступать к их выполнению.
