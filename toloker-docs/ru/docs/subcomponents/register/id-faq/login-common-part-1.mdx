Воспользуйтесь рекомендациями Справки [Яндекс ID](https://yandex.ru/support/passport/troubleshooting/phone-problems.html#phone-problems__no-code). Если они не помогли решить проблему, напишите в [службу поддержки](https://yandex.ru/support/passport/troubleshooting/nosms-form.html).

<Cut title="Не могу зарегистрироваться, потому что номер телефона уже привязан к другой учетной записи">

Номер телефона может быть привязан только к одному аккаунту в Толоке. Cкорее всего, вы ранее регистрировались в системе под другим логином. Вспомните логин, с которым вы регистрировались, и используйте для входа именно его. Если не можете вспомнить, воспользуйтесь [Восстановлением доступа](https://passport.yandex.ru/passport?mode=restore).

Согласно [Пользовательскому соглашению](https://join.toloka.ai/legal/user-agreement-ru/):

- пользователь может иметь только один аккаунт в Толоке;

- повторная регистрация запрещена;

- у пользователя должен быть уникальный номер телефона.

</Cut>

<Cut title="Как поменять номер телефона в моем аккаунте">

Перейдите в Яндекс ID и измените основной номер. После смены номера телефона вы не сможете выводить деньги в течение недели.

<Note mode="note" title="Примечание">

Если старый номер вам недоступен, изменение номера займет месяц. Подробнее об изменении номера телефона см. [Справку Яндекс ID](https://yandex.ru/support/passport/authorization/phone.html#change).

</Note>

</Cut>

<Cut title="Где посмотреть свой логин в веб-версии Толоки">

В веб-версии Толоки логин указан в верхнем правом углу, рядом с аватаркой. Если рядом с аватаркой отображаются ваше имя или ник, нажмите на аватарку — логин будет указан в открывшемся окне.

</Cut>
