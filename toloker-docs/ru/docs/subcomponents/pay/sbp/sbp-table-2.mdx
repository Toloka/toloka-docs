Средства выводятся на счет в банке. Список доступных банков зависит от оператора услуги и может меняться.

<Note mode="restriction" title="Внимание">

Вывод средств через СБП доступен только для граждан РФ, подтвердивших статус [самозанятого](../self-employed/about) и указавших Толоку партнером в системе Мой Налог.

</Note>
