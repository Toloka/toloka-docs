3. Заполните анкету: введите данные, прикрепите фотографии документов и селфи. Затем нажмите кнопку **Отправить на модерацию**.

Мы обработаем анкету в течение 3–5 дней. После этого вам будет нужно [подтвердить статус самозанятого](../accept-status) в Толоке.
