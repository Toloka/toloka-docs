# Оплата налогов

Ваши данные о заработке автоматически отправляются в Налоговую службу. С вашего заработка начисляется [налог на профессиональный доход](https://npd.nalog.ru/) по ставке 6%.

**Налог вы оплачиваете сами.** При выводе средств эта сумма не удерживается нами.

Сумма к оплате отобразится в [личном кабинете](https://lknpd.nalog.ru/auth/login) в системе **Мой налог**. Оплатить налог нужно не позднее 25 числа следующего месяца. Можно оплатить картой или по квитанции.
