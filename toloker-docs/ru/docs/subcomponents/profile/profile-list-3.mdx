1. В левой части страницы **Настройки** нажмите кнопку **Сайты**.

1. В блоке **Уведомления** нажмите кнопку **От прочих сайтов**.

1. На вкладке **Разрешена** нажмите кнопку **Добавить** вверху справа.

1. Введите адрес `we.toloka.ai`.

1. Нажмите **Добавить**.
