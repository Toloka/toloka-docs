# Профиль

В вашем профиле должны быть указаны ваши настоящие имя и фамилия. Это требуется для [вывода денег](../pay/about) из [Толоки](https://toloka.ai/tolokers/ru/) в платежную систему. Если при регистрации вы указали другое имя, отредактируйте профиль.
