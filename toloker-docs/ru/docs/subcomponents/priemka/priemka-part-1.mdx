# Задания на проверке

На проверку некоторых заданий требуется время. Заказчик может отклонить задание, если оно выполнено не по инструкции. Отклоненные задания не оплачиваются.

## Сколько времени занимает проверка {#deadline}

Проверка может занять несколько дней. Срок проверки указан в поле **На проверке до** на странице **История**. Все задания проверяются в указанные сроки, хотя зачастую проверка занимает меньше времени.
