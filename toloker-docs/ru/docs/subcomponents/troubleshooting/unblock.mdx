# Восстановление доступа

Если вы получили письмо о блокировке вашего аккаунта в [Толоке](https://toloka.ai/tolokers/ru/) и вам предложено восстановить доступ подтверждением личности, заполните форму.

Для подтверждения личности прикрепите свое фото с одним из следующих документов в руках:

- паспорт;
- заграничный паспорт;
- водительское удостоверение;
- ID-карта, если она используется в вашей стране.

На документе должны быть хорошо видны фамилия, имя и отчество, фотография и дата рождения.
