# О приложении

[Толока](https://toloka.ai/tolokers/ru/) — это краудсорсинговая платформа, которая позволяет обрабатывать большие массивы данных, распределяя работу среди множества исполнителей.

Авторы размещают проекты и устанавливают цену, а исполнители выбирают проект и [выполняют его](../tasks) за [вознаграждение](../priemka#pay).

Проекты могут быть разными:

- Полевые проекты. Вы получаете точку на карте, куда нужно прийти, например, найти организацию (кафе, магазин и т. д.), узнать режим работы, сделать фотографии.

- Обычные проекты. Чтобы выполнить их, не нужно ехать или идти куда-либо.

Выполненные сеты заданий [проверяются](../priemka) заказчиком. По результатам проверки заказчик оценивает ваши [навыки](../skills).

Стоимость проектов определяется заказчиками. Вы можете [вывести средства](../pay/about) различными способами в зависимости от вашей страны.
