Чтобы написать заказчику:

1. Откройте страницу [Сообщения](http://toloka.yandex.ru/messages).

1. Нажмите кнопку **Написать заказчику** и выберите получателя.

1. В теме письма укажите название задания.

1. Напишите сообщение и нажмите кнопку **Отправить**.
