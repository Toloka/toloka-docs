# Список заданий

Откройте страницу [Задания](https://toloka.yandex.ru/tasks). По умолчанию задания отображаются по критерию **сначала избранные**. Вы можете установить другой порядок сортировки в строке сверху:

- **сначала новые**
- **по цене**
- **сначала избранные**

Также вы можете фильтровать задания по категориям (с обучением, с отложенной приёмкой, скрытые и т.д.) и по заказчикам.

Недоступные задания отмечены значком <img src="https://yastatic.net/s3/doc-binary/src/support/toloka-tolokers/ru/lock.svg" alt="" />. Рядом указано условие доступа к заданию. Например, условие «мобильное приложение» означает, что задание нельзя выполнить в десктопной версии Толоки, только в мобильном приложении.

## Избранные и скрытые задания {#selected-n-hidden}

Вы можете добавить задание в избранные или скрыть его в списке.

**Добавить задание в избранные**

- Наведите курсор на карточку задания и нажмите кнопку <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1677847613863/1677847613863_original_svg.svg" alt="" /> справа.

- Выберите Добавить в избранные.

    Или нажмите кнопку <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1677848154081/1677848154081_original_svg.svg" alt="" /> рядом с названием задания.

Нажмите сначала избранные, чтобы избранные задания выводились в начале списка. Если задания в списке избранных неактивны, возможно, они закончились или вы допускали много ошибок и заказчик ограничил вам доступ к ним. Когда задания появятся, они снова будут активны.

<Note mode="tip" title="Совет">

В Толоке много заданий, и каждый день появляются новые. Поэтому регулярно проверяйте список заданий, чтобы подобрать себе что-нибудь интересное.

</Note>

**Скрыть задание**

- Наведите курсор на карточку задания и нажмите кнопку <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1677847613863/1677847613863_original_svg.svg" alt="" /> справа.

- Выберите Скрыть задание.

- Выключите фильтр Скрытые.

**Убрать задание из скрытых**

- Включите фильтр Скрытые.

- Наведите курсор на карточку задания и нажмите кнопку <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1677847613863/1677847613863_original_svg.svg" alt="" /> справа.

- Выберите Показать задание.
