2. Откройте страницу [**Профиль**](https://toloka.yandex.ru/profile) в Толоке. В верхней части будет окно с предложением подтвердить свою личность. Нажмите кнопку **Подтвердить**.

    <Image width="700px" withBackground imageId="c5225fcc353ccf43b44b553364d88786" alt="" />

    Если в профиле вы не видите этого окна — возможно, вы зарегистрировались недавно и выполнили мало оплачиваемых заданий. Анкета открывается пользователям, которые заработали в Толоке больше 1&nbsp;$. Выполните задания до необходимой суммы или напишите в [службу поддержки](../../troubleshooting/support/?form-source=self-employed-send-request) и попросите, чтобы анкету открыли вручную.
