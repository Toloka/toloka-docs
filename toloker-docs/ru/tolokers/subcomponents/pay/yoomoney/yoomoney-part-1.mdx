# ЮMoney

#### Особенности ЮMoney

ЮMoney — платежная система, через которую могут выводить средства пользователи Толоки. В ЮMoney легко создать кошелек и идентифицировать его.

<Note mode="restriction" title="Внимание">

Вывод средств через ЮMoney доступен только для граждан РФ, подтвердивших статус [самозанятого](../self-employed/about) и указавших Толоку партнером в системе Мой Налог.

</Note>
