# Selecting a project

Select:

- [Non-field project](#non-walk)
- [Field project](#walk)
- [Tips and recommendations](#android-list)

## Regular (non-field) projects `{#common}`

To complete a non-field project:

1. Open the project feed by tapping **<img src="https://toloka.ai/files/hamburger-menu.svg" alt="" /> → Available**.

1. Read the instructions and go to the project.

1. Tap **Start**.

1. Answer the questions and tap **Done**.

You will be redirected to a page with the next project. To exit the project, tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733319521/1676733319521_original_svg.svg" alt="" /> at the top of the screen and choose **Finish**.

## How to save uncompleted projects `{#howtosave}`

You can save projects that you didn't complete in **Reserved**. To do this:

1. Tap <img src="https://toloka.ai/files/hamburger-menu.svg" alt="" /> → **Settings**.

1. Select the **Always reserve task sets to complete later** option

To continue completing the project, tap <img src="https://toloka.ai/files/hamburger-menu.svg" alt="" /> → **Reserved**, find the project you need, and tap **Resume**.

## Field projects `{#fieldwork}`

To complete a field project:

1. Open the project feed. To do this, tap <img src="https://toloka.ai/files/hamburger-menu.svg" alt="" /> → **Available**.

1. Use one of these ways to open project locations:

    * Select a project in the project feed and tap **Open map** to see the locations for this project.

    * Open the common map by tapping <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676448795481/1676448795481_original_png.png" alt="" /> at the bottom of the screen to view locations with available projects.

    To return to the project feed, tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676448711664/1676448711664_original_png.png" alt="" /> at the bottom of the screen.

1. Tap a placemark with a project.

    If there are several projects in one spot, use the search function. To find a project in the list:

    1. Pull the list up.

    1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733320430/1676733320430_original_svg.svg" alt="" /> on the top right of the screen.

    1. Start entering the project name or description.

    To set a minimum price per project:

    1. Tap <img src="https://toloka.ai/files/filters-menu.svg" alt="" /> at the bottom of the screen.

    1. Move the slider.

    1. Tap **Apply**.

1. Read the instructions.

1. Reserve a task set by tapping **Reserve**. The task set is displayed on the **Reserved** page. The countdown starts from this moment. The timer is displayed in the task set header next to the <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733320837/1676733320837_original_svg.svg" alt="" /> icon. Max duration is specified in the task set card.

    You can reserve several task sets at once.

1. To build a route, tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733319521/1676733319521_original_svg.svg" alt="" /> → **Get directions**. The route is created in the [Yandex Maps](https://itunes.apple.com/us/app/yandex-maps/id313877526) app.

1. Go to the location shown in the task set. Next, select the desired task set on the **Reserved** and tap **Continue**.

1. Complete the project:

    * Re-read the instructions.

    * Answer the questions and attach a photo (if required).

    * Tap **Done**.

        <Note mode="warning" title="Warning">

        You need to tap **Done** while you are at the location shown in the project. Otherwise, your response might be rejected.

        </Note>

The project with the responses is saved on the **Done** page until you connect to Wi-Fi, if you set the **Submit over Wi-Fi only** option in the [settings](../settings). You need to make sure the project is submitted before the project time expires.

### How do I make changes to a task set? `{#makechanges}`

If a task set is completed but isn't sent yet, you can make changes to it:

1. Open the task set on the **Done** page.

1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733319521/1676733319521_original_svg.svg" alt="" /> at the top of the screen and select **Change status to reserved**.

1. Go to the **Reserved** page and make changes to the task set.

1. Tap **Done**.

### How do I submit a task set? `{#sendtask}`

Task set responses are sent to the requester as soon as you have internet access. To send, you need to open the mobile app. If the **Submit over Wi-Fi only** option is chosen in the [settings](../settings), the task set is sent after you connect to Wi-Fi. To submit a task set over mobile internet:

1. Open the task set on the **Done** page.

1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733319521/1676733319521_original_svg.svg" alt="" /> at the top of the screen and choose **Submit**.

## Tips and recommendations `{#tips}`

The project feed displays all projects, including both field and non-field projects. To open the list of projects, tap <img src="https://toloka.ai/files/hamburger-menu.svg" alt="" /> → **Available**.

To manage the list of projects, apply filters and configure sorting:

1. Tap <img src="https://toloka.ai/files/filters-menu.svg" alt="" /> at the top of the screen.

1. Choose one or more filters. Choose the requesters you want to see in the list. Set the sorting type.

1. Tap **Apply**.

Unavailable projects are marked with the <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733315411/1676733315411_original_svg.svg" alt="" /> icon. The condition for accessing the project is indicated next to it. Some projects can only be performed in the browser on the [toloka.ai/tolokers/en/](https://toloka.ai/tolokers/en/) website.

### How to add a project to favorites `{#favourites}`

Mark the projects that you liked:

1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733323602/1676733323602_original_svg.svg" alt="" /> in the project card.

2. Select **Add to favorites**.

To place your favorite projects at the top of the list, set the **Favorites first** [sorting](#android-list/list-settings-android) type.

### How do I hide a project? `{#hide}`

Hide a project so that it doesn't appear in the list.

1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733323602/1676733323602_original_svg.svg" alt="" /> in the project card.

1. Select **Hide project**.

### How to change pin appearance `{#changelook}`

To choose which project information to display in the label:

1. Tap <img src="https://toloka.ai/files/hamburger-menu.svg" alt="" /> → **Settings**.

1. Open **Change pin appearance** and choose the desired option.

1. Tap **Save**.
