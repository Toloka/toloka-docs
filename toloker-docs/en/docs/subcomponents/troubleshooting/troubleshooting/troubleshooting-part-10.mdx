<Cut title="Can several Tolokers use the same IP address?">

According to the [User Agreement](https://toloka.ai/tolokers/legal/user-agreement/), a user can only have one account in [Toloka](http://toloka.yandex.com/). Users who have different accounts in Toloka can complete tasks from the same IP address.

</Cut>

<Cut title="Can I install a third-party program or extension to refresh the page in the browser, notify me about new tasks, and so on?">

We can't guarantee that [Toloka](http://toloka.yandex.com/) will work correctly with additional programs or browser extensions because they were not tested by us. They might be used by hackers to access your account.

</Cut>
