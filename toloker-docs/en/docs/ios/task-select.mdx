# Selecting a project

## Regular (non-field) projects `{#common}`

To complete a non-field project:

1. Open the project feed by tapping the **All projects** button.

1. Read the instructions and go to the project.

1. Tap **Start**.

1. Answer the questions and tap **Submit**.

You will be redirected to a page with the next project. To exit the project, tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733323602/1676733323602_original_svg.svg" alt="" /> at the bottom of the screen and choose **Finish**.

## Field projects `{#fieldwork}`

To complete a field project:

1. Open the project feed. To do this, tap **Projects**.

1. Use one of these ways to open project locations:

    * Select a project in the project feed and tap **Open map** to see the locations for this project.

    * Open the common map by tapping **Map** to view locations with available projects.

    <Note mode="tip" title="Tip">

    To save mobile data, download the map of the region to your device. To do this, tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676448782111/1676448782111_original_png.png" alt="" />.

    To delete the downloaded maps, open **More** → **Settings** → **Delete downloaded maps**.

    </Note>

    To return to the project feed, tap the **List** button at the top of the screen.

1. Tap a placemark with a project.

    To set a minimum price per project:

    1. Tap **Filters** at the top of the screen.

    1. Move the slider.

    1. Tap **Done**.

1. Read the instructions.

1. Reserve a task set by tapping **Reserve**. The task set is displayed on the **My projects** page. The countdown starts from this moment. The timer is displayed in the task set header next to the <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733320837/1676733320837_original_svg.svg" alt="" /> icon. Max duration is specified in the task set card.

    You can reserve several task sets at once.

1. Create a route to the location by tapping **Get directions**. The route is created in the [Yandex Maps](https://itunes.apple.com/us/app/yandex-maps/id313877526) app.

1. Go to the location shown in the project. Next, select the desired project on the **My projects → Active** page and tap **Continue**.

1. Complete the project:

    * Re-read the instructions.

    * Answer the questions and attach a photo (if required).

    * Tap **Submit**.

        <Note mode="warning" title="Warning">

        You need to tap **Submit** while you are at the location shown in the project. Otherwise, your response might be rejected.

        </Note>

The task sets with the responses is saved on the **My projects → Done** page until you connect to Wi-Fi, if you set the **Submit over Wi-Fi only** option in the [settings](../settings). You need to make sure the task set is submitted before the task set time expires.

### How do I make changes to a task set? `{#makechanges}`

If a task set is completed but isn't sent yet, you can make changes to it:

1. Open the task set on the **My projects → Done** page.

1. Tap **Change to Active**.

1. Go to the **My projects → Active** page and tap **Continue**.

1. Make changes to the task set and tap **Submit**.

### How do I submit a task set? `{#sendtask}`

Task set responses are sent to the requester as soon as you have internet access. To send, you need to open the mobile app. If the **Submit over Wi-Fi only** option is chosen in the [settings](../settings), the task set is sent after you connect to Wi-Fi.

To submit a task set over mobile internet:

1. Open the task set on the **My projects → Done** page.

1. Tap **Send now**.

## Tips and recommendations `{#tips}`

The project feed displays all projects, including both field and non-field projects. To open the list of projects, tap **Projects**.

To manage your project feed:

1. Tap **Filters** at the top of the screen.

1. Use filters or sorting tools:

    <Definition title="Project price">

    To set the minimum project price, move the slider.

    </Definition>

    <Definition title="Project categories">

    Enable or disable categories such as **Field project** or **with training**.

    </Definition>

    <Definition title="Requesters">

    Tap a requester to add their projects to the list or exclude them from the list.

    </Definition>

    <Definition title="Sorting">

    Choose the sorting type: by price, last updated, or Favorites.

    </Definition>

1. Tap **Done** at the top of the page.

Unavailable projects are marked with the <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733315411/1676733315411_original_svg.svg" alt="" /> icon. The condition for accessing the project is indicated next to it. Some projects can only be completed in the browser on the [toloka.ai/tolokers/en/](https://toloka.ai/tolokers/en/) website.

### How to add a project to favorites `{#favourites}`

Mark the projects that you liked:

1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733323602/1676733323602_original_svg.svg" alt="" /> in the project card.

2. Select **Add to favorites**.

To place your favorite projects at the top of the list, set the **Favorites first** [sorting](#android-list/list-settings-ios) type.

### How do I hide a project? `{#hide}`

Hide a project so that it doesn't appear in the list.

1. Tap <img src="https://tlkfrontprod.azureedge.net/portal-production/static/uploaded/images/1676733323602/1676733323602_original_svg.svg" alt="" /> in the project card.

1. Select **Hide project**.

### How to change pin appearance `{#changelook}`

To choose which project information to display in the label:

1. Tap **More → Settings**.

1. Open **Change pin appearance** and choose the desired option.
