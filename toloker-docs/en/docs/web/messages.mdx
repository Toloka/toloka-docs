- If a task doesn't work correctly (it shows a blank screen, doesn't display images, the **Submit** button doesn't work, and so on).

- To ask about the instructions, training or the skill level.

To write to the requester:

1. Open the [Messages](https://we.toloka.ai/messages) page.

1. Tap **Contact requester** and choose the recipient.

1. Enter the task name in the subject.

1. Write a message and tap **Send**.
