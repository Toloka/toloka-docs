## Path parameters {#path-params}

Parameter | Overview
----- | -----
**thread_id** | Message thread ID.

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.
**Content-Type** | Specifies the data format in the request body.

## Request body {#body}

```json
{
  "text": {
    "EN": "Thank you!"
  }
}
```

Parameter | Overview
----- | -----
**text** | **string**<br/><br/>Message text. You can enter the text in multiple languages (the message is sent in the Toloker's language). Format: `"<language RU/EN/TR/ID/FR>": "<message text>"`.
