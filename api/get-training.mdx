# Get training properties

<Announce/>

Gets the properties of a training pool.

You can get the training pool ID from the [list of training pools](/docs-new/api/get-training-list).

<Tabs tabs={[

## Path parameters {#path-params}

Parameter | Overview
----- | -----
**training_id** | Pool ID.

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Response {#response}

Contains information about a training pool (see the description in the [Create a training pool](/docs-new/api/create-training#response) section).
