**string**<br/><br/>Parameters for sorting in ascending order:

- `id` — ID of the training pool.
- `created` — The date when the training pool was created, in UTC using ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
- `last_started` — The date when the training pool was last started, in UTC using ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.<br/><br/>To learn how to configure sorting, see [Sorting the list of objects](/docs/api/sorting).
