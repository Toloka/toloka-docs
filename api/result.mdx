<Announce/>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.

## Query parameters {#query-params}

Specified in the link after the question mark; separated by `&`.

Parameter | Overview
----- | -----
**status** | <ResultTable1/>
**task_id** | **string**<br/><br/>You must specify either `task_id`,`pool_id`, or `task_suite_id`.<br/><br/>The task ID in the suites that were generated automatically using "smart mixing". You will get responses for task suites that contain the specified task.
**task_suite_id** | **string**<br/><br/>You must specify either `task_id`,`pool_id`, or `task_suite_id`.<br/><br/>ID of a task suite.
**pool_id** | **string**<br/><br/>You must specify either `task_id`,`pool_id`, or `task_suite_id`.<br/><br/>Pool ID.
**user_id** | **string**<br/><br/>Toloker ID.
**sort** | <ResultTable2/>
**Standard query parameters** | [limit](./standard-query-parameters#limit), [id_gt](./standard-query-parameters#id_gt), [id_gte](./standard-query-parameters#id_gte), [id_lt](./standard-query-parameters#id_lt), [id_lte](./standard-query-parameters#id_lte), [created_gt](./standard-query-parameters#created_gt), [created_gte](./standard-query-parameters#created_gte), [created_lt](./standard-query-parameters#created_lt), [created_lte](./standard-query-parameters#created_lte), [submit_gt](./standard-query-parameters#submit_gt), [submit_gte](./standard-query-parameters#submit_gte), [submit_lt](./standard-query-parameters#submit_lt), [submit_lte](./standard-query-parameters#submit_lte), [acceptedt_gt](./standard-query-parameters#accepted_gt), [accepted_gte](./standard-query-parameters#accepted_gte), [accepted_lt](./standard-query-parameters#accepted_lt), [accepted_lte](./standard-query-parameters#accepted_lte), [rejected_gt](./standard-query-parameters#rejected_gt), [rejected_gte](./standard-query-parameters#rejected_gte), [rejected_lt](./standard-query-parameters#rejected_lt), [rejected_lte](./standard-query-parameters#rejected_lte), [skipped_gt](./standard-query-parameters#skipped_gt), [skipped_gte](./standard-query-parameters#skipped_gte), [skipped_lt](./standard-query-parameters#skipped_lt), [skipped_lte](./standard-query-parameters#skipped_lte), [expired_gt](./standard-query-parameters#expired_gt), [expired_gte](./standard-query-parameters#expired_gte), [expired_lt](./standard-query-parameters#expired_lt), [expired_lte](./standard-query-parameters#expired_lte).
