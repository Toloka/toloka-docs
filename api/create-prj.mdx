# Create a project

<Announce/>

Creates a project.

<Note mode="restriction" title="Restriction">

You can send a maximum of 20 requests of this kind per minute and a maximum of 100 requests per day.

</Note>

Parameter | Overview
----- | -----
**public_name** | **string \| required**<br/><br/>Name of the project. It will be shown to Tolokers.
**public_description** | **string \| required**<br/><br/>Description of the project. It will be shown to Tolokers.
**task_spec** | **object \| required**<br/><br/>Parameters for input and output data and the task interface.
**task_spec.input_spec** | **object \| required**<br/><br/>The input data parameters for tasks. The complete list of parameters is shown in the [Input and output data](#in-out) table.
**task_spec.output_spec** | **object \| required**<br/><br/>Parameters for output data from the input fields. The complete list of parameters is shown in the [Input and output data](#in-out) table.
**task_spec.view_spec** | **object**<br/><br/>Description of the task interface. The complete list of parameters is shown in the [Task interface](#view-spec-section) table.
**assignments_issuing_view_config** | **string \| required if**<br/><br/>Required if `assignments_issuing_type=MAP_SELECTOR`.<br/><br/>Settings for displaying field tasks. For a complete list of parameters, see [Settings for displaying field tasks](#assignments-issuing-view-config-section).
**public_instructions** | **string**<br/><br/>Instructions for completing the task. You can use any HTML markup in the instructions.
**private_comment** | **string**<br/><br/>Comments that are only visible to the requester.
**assignments_issuing_type** | <CreatePrjTable1/>
**assignments_automerge_enabled** | **boolean**<br/><br/>Resolve [merging identical tasks](/docs-new/api/tasks#task-merge) in the project. The default value is `false`.
**max_active_assignments_count** | **integer**<br/><br/>The number of task suites the Toloker can complete simultaneously ("Active" status).
**quality_control** | **object**<br/><br/>The quality control rule.
**quality_control.configs[]** | **array of objects**<br/><br/>[Presets](/docs-new/api/quality-control)
**localization_config** | **object**<br/><br/>Block of translations to other languages. The complete list of parameters is shown in the [Translations to other languages](#localization-config-section) table.<br/><br/>For more information about translation, see [Translations to other languages](https://toloka.ai/en/docs/guide/concepts/project-languages).

## Input and output data (input_spec and output_spec) {#in-out}

The `input_spec` and `output_spec` parameters contain JSON with the input data properties and response validation parameters. Use them to define the data type (string, integer, URL, and so on) and specify validation parameters (such as the string length).

Parameter | Overview
----- | -----
**<ID\>** | <CreatePrjTable2/>
**type** | <CreatePrjTable3/>
**required** | **boolean**<br/><br/>Whether the object or input field is required. The default value is `true`.
**hidden** | **boolean**<br/><br/>Whether to hide field with the input value from the Toloker or not. The default value is `false`.<br/><br/>For output data, always `false`.
**min_value** | **float**<br/><br/>Minimum value of the number.
**max_value** | **float**<br/><br/>Maximum value of the number.
**allowed_values[]** | **array of strings, array of integers, array of floats**<br/><br/>Allowed values.<br/><br/>Setting acceptable values improves the quality of [response aggregation](/docs-new/api/aggregated-solutions).
**min_length** | **integer**<br/><br/>Minimum length of the string.
**max_length** | **integer**<br/><br/>Maximum length of the string.
**current_location** | **string**<br/><br/>Only for output data of the `coordinates` type: populate the field with the Toloker's current coordinates in the field (`true`/`false`). Used in tasks for the mobile app.

## Response {#response}

Contains information about the uploaded project in JSON format. The project is automatically assigned an ID.

It includes:

- [Parameters that are used when creating a project](#prj-param).
- Parameters that are assigned automatically.

Parameter | Overview
----- | -----
**owner** | **object**<br/><br/>Parameters of the requester that created the project
**owner.id** | **string**<br/><br/>Parameters to sort by:
**owner.myself** | <CreatePrjTable13/>
**id** | **string**<br/><br/>Project ID (assigned automatically).
**status** | <CreatePrjTable14/>
**created** | **string**<br/><br/>The UTC date and time the project was created, in ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
