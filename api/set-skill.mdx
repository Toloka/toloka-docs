# Set the skill value

<Announce/>

Sets the skill value.

<Note mode="restriction" title="Restriction">

You can send a maximum of 100,000 requests of this kind per day.

</Note>

## Request {#request}

<Tabs tabs={[
  { title: 'Production version', value: <SetSkillList1/> },
  { title: 'Sandbox', value: <SetSkillList2/> },
]}/>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.
**Content-Type** | Specifies the data format in the request body.

## Request body {#body}

```json
{
  "skill_id": 32,
  "user_id": "566ec2b0ff0deeaae5f9d500",
  "value": 59.7,
  "reason": "High-quality performance of tasks"
}
```

Parameter | Overview
----- | -----
**skill_id** | **string \| required**<br/><br/>Skill ID.
**user_id** | **string \| required**<br/><br/>Toloker ID.
**value** | **float \| required**<br/><br/>Fractional value of the skill. Minimum — 0, maximum — 100.
**Reason** | **string**<br/><br/>The reason for changing or assigning a skill.

## Response {#response}

Updated skill value.

```json
{
  "id": "43",
  "skill_id": "32",
  "user_id": "566ec2b0ff0deeaae5f9d500",
  "value": 60,
  "exact_value": 59.7,
  "created": "2021-06-17T09:30:11.681",
  "modified": "2021-05-19T10:52:22.130"
}
```

Parameter | Overview
----- | -----
**id** | **string**<br/><br/>ID of the Toloker's skill, assigned to the "skill-user" pair.
**skill_id** | **string**<br/><br/>Skill ID.
**user_id** | **string**<br/><br/>Toloker ID.
**value** | **integer**<br/><br/>The integer value of that skill (`exact_value`, rounded to the nearest integer). Minimum — 0, maximum — 100.
**exact_value** | **float**<br/><br/>Fractional value of the skill. Minimum — 0, maximum — 100.
**created** | **string**<br/><br/>The date and time in UTC when the Toloker was first assigned the skill with the specified value. It uses ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
**modified** | **string**<br/><br/>The date and time in UTC when the Toloker's skill level changed. It uses ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
