**string**<br/><br/>Parameters to sort by:

- `id` — Pool ID.
- `created` — The date when the pool was created, in UTC using ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
- `last_started` — The date when the pool was last started, in UTC using ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.<br/><br/>To learn how to configure sorting, see [Sorting the list of objects](/docs-new/api/sorting).
