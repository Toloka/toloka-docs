# Edit a training pool

<Announce/>

Makes changes to a training pool.

</Note>

## Path parameters {#path-params}

Parameter | Overview
----- | -----
**training_id** | Pool ID.

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.
**Content-Type** | Specifies the data format in the request body.

## Request body {#body}

In the request body, specify **all** the pool [parameters](/docs-new/api/create-training#training-param), including those that are updated.

## Response {#response}

Contains information about a modified training pool (see the description in the [Create a training pool](/docs-new/api/create-training#response) section).
