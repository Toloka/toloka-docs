# Aggregate responses to a single task

<Announce/>

Starts aggregating responses to a single task.

## Request {#request}

<Tabs tabs={[
  { title: 'Production version', value: <AggregateByTaskList1/> },
  { title: 'Sandbox', value: <AggregateByTaskList2/> },
]}/>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.
**Content-Type** | Specifies the data format in the request body.

## Request body {#body}

```json
{
  "task_id": "afd1234d-12314a-cfd1424d-31214b",
  "pool_id": "283",
  "type": "WEIGHTED_DYNAMIC_OVERLAP",
  "answer_weight_skill_id": "1289",
  "fields": [
    {
      "name": "result"
    }
  ]
}
```

Parameter | Overview
----- | -----
**task_id** | **string**<br/><br/>Task ID.
**pool_id** | **string**<br/><br/>Pool ID.
**type** | <AggregateByTaskTable1/>
**answer_weight_skill_id** | **string**<br/><br/>A skill that determines the weight of the Toloker's response.
**fields** | **object**<br/><br/>[Output data fields](/docs/guide/result-aggregation) to use for aggregating responses. For best results, each of these fields must have a limited number of response options.
**fields.name** | **string**<br/><br/>The output data field name.

## Response {#response}

Contains the aggregated task response.

```json
{
  "confidence": 0.937152,
  "output_values": {
    "result": "OK"
  },
  "pool_id": "283",
  "task_id": "afd1234d-12314a-cfd1424d-31214b"
}
```

Parameter | Overview
----- | -----
**confidence** | **integer**<br/><br/>Confidence in the aggregate response.
**output_values** | **object**<br/><br/>Output data fields and aggregate response.
**pool_id** | **string**<br/><br/>Pool ID.
**task_id** | **string**<br/><br/>Task ID.
