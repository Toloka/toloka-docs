import MessageSendTable10 from './message-send-table-10.mdx'
import MessageSendTable11 from './message-send-table-11.mdx'

```json
  {
    "topic": {
      "EN": "You have got a bonus!"
    },
    "text": {
      "EN": "The bonus was awarded for good job!"
    },
    "recipients_select_type": "FILTER",
    "recipients_filter": {
      "and": [
        {
          "category": "skill",
          "key": "2022",
          "operator": "GT",
          "value": 90
        }
      ]
    },
    "answerable": true
  }
```

Parameter  | Overview
----- | -----
**topic** | **object \| required**<br/><br/>    Subject of the message. You can enter the subject in multiple languages (the message is sent in the Toloker's language). Format: `"<language RU/EN/TR/ID/FR>": "<title text>"`.
**answerable** | <MessageSendTable10/>
**text** | **object \| required**<br/><br/>    Message text. You can enter the text in multiple languages (the message is sent in the Toloker's language). Format: `"<language RU/EN/TR/ID/FR>": "<message text>"`.
**recipients_select_type** | <MessageSendTable11/>
**recipients_filter** | **object \| required if**<br/><br/>    Required if `recipients_select_type=FILTER`.<br/><br/>    [Filter](/docs-new/api/filters) for selecting recipients.
