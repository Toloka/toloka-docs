**string**<br/><br/>The scope of the ban:

- `ALL_PROJECTS` — All the requester's projects.
- `PROJECT` — A single project (specify the `project_id`).
- `POOL` — A pool (specify the `pool_id`).
- `SYSTEM` — Users that performed the requester's tasks but were [banned on the platform](/docs-new/guide/ban#ban__ban-platform).
