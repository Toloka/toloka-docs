# Edit a pool

<Announce/>

Makes changes to a pool.

The pool can't be edited if it's open. [Close the pool](/docs-new/api/close-pool-for-update) before editing.

<Note mode="note" title="Note">

Learn about editing a training pool in the section [Edit a training pool](/docs-new/api/edit-training).

</Note>

## Headers {#headers}

Title | Overview
----- | -----
**Authorization** | A token for account authorization. Add OAuth as a prefix.
**Content-Type** | Specifies the data format in the request body.

## Request body {#body}

In the request body, specify **all** the pool [parameters](/docs-new/api/create-pool#pool-param), including those that are updated.

## Response {#response}

Contains updated information about the pool (see the description in the [Create a pool](/docs-new/api/create-pool#response) section).
