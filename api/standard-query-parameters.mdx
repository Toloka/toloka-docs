# Standard query parameters

Parameter | Overview
----- | -----
**limit** | **integer**<br/><br/>Limit on the number of results to return. By default — 50; maximum — 300.
**id_gt** | **string**<br/><br/>Objects with an ID greater than the specified value.
**id_gte** | **string**<br/><br/>Objects with an ID greater than or equal to the specified value.
**id_lt** | **string**<br/><br/>Objects with an ID less than the specified value.
**id_lte** | **string**<br/><br/>Objects with an ID less than or equal to the specified value.
**overlap_gt** | **integer**<br/><br/>Objects with an overlap greater than the specified value.
**overlap_gte** | **integer**<br/><br/>Objects with an overlap greater than or equal to the specified value.
**overlap_lt** | **integer**<br/><br/>Objects with an overlap less than the specified value.
**overlap_lte** | **integer**<br/><br/>Objects with an overlap less than or equal to the specified value.
**task_id_gt** | **string**<br/><br/>Tasks with an ID greater than the specified value.
**task_id_gte** | **string**<br/><br/>Tasks with an ID greater than or equal to the specified value.
**task_id_lt** | **string**<br/><br/>Tasks with an ID less than the specified value.
**task_id_lte** | **string**<br/><br/>Tasks with an ID less than or equal to the specified value.

</Note>
