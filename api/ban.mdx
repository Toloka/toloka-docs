# Overview

You can block a Toloker's access to one or more projects. This lets you control which Tolokers will complete tasks. For example, you can select Tolokers with a skill value lower than N and block their access to tasks. You can also unblock access.

## Methods {#methods}

Method | Endpoint | Overview
----- | ----- | -----
PUT | [/user-restrictions](/docs-new/api/ban-create) | Blocks a Toloker from accessing tasks.
GET | [/user-restrictions](/docs-new/api/ban-get-list) | Gets the list of Toloker bans.
GET | [/user-restrictions/<ban_id>](/docs-new/api/ban-get-info) | Gets information about a ban.
DELETE | [/user-restrictions/<ban_id>](/docs-new/api/ban-delete) | Removes a ban.
