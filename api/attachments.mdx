# Overview

Files uploaded by Tolokers are saved in Toloka.

## Methods {#methods}

Method | Endpoint | Overview
----- | ----- | -----
GET | [/attachments](/docs-new/api/get-attachment-list) | Gets the list of files saved in Toloka.
GET | [/attachments/<file_id>](/docs-new/api/get-attachment) | Gets the properties of a file attached to a task response.
GET | [/attachments/<file_id>/download](/docs-new/api/download-attachment) | Downloads a file attached to a task response.
