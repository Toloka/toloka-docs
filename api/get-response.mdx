# Overview

You can get:

- Responses for the pool's task suites and their properties.
- [Files uploaded by Tolokers](/docs-new/api/attachments).
- [Aggregated responses](/docs-new/api/aggregated-solutions).

## Methods {#methods}

Method | Endpoint | Overview
----- | ----- | -----
GET | [/assignments](/docs-new/api/result) | Gets responses for all the pool's task suites.
GET | [/assignments/<response_id>](/docs-new/api/get-assignment-id) | Gets the properties of this response.
