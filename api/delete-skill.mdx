## Path parameters {#path-params}

Parameter | Overview
----- | -----
**id** | The ID of a Toloker's skill that is assigned to the "skill-Toloker" pair. You can find this ID using the [Get a list of Tolokers who have skills](/docs/api/get-user-skill-list) request.
