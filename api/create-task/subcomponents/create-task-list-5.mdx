Parameter | Overview
----- | -----
**items** | **object**<br/><br/>  An object with the created tasks.
**<n\>** | **object**<br/><br/>  Sequential number of the task in the array when created (starting from 0).
**id** | **string**<br/><br/>  Task ID.
**pool_id** | **string**<br/><br/>  The ID of the pool that the task is uploaded to.
**overlap** | **string**<br/><br/>  Task suite overlap.
**created** | **string**<br/><br/>  The UTC date and time when the task suite was created, in ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
**validation_errors** | **object**<br/><br/>  An object with errors in tasks. Returned if the request has the parameter `skip_invalid_items=true`.
