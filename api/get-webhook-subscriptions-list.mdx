## Response {#response}

Contains a list of subscriptions with information about each of them in JSON format.

```json
{
  "items": [
    {
      "webhook_url": "https://awesome-requester.com/toloka-webhook",
      "event_type": "ASSIGNMENT_CREATED",
      "pool_id": "121212",
      "id": "webhook-subscription-1",
      "created": "2020-02-03T15:00:00"
    },
    {
      "webhook_url": "https://awesome-requester.com/toloka-webhook",
      "event_type": "POOL_CLOSED",
      "pool_id": "121212",
      "id": "webhook-subscription-2",
      "created": "2020-02-03T15:00:00"
    }
  ],
  "has_more": false
}
```

Parameter | Overview
----- | -----
**items[]** | **array of objects**<br/><br/>An array of objects with information about the requested subscriptions.
**webhook_url** | **string**<br/><br/>The URL that notifications will be sent to.
**event_type** | <GetWebhookSubscriptionsListTable3/>
**pool_id** | **string**<br/><br/>ID of the pool that the subscription was created for.
**id** | **string**<br/><br/>ID of the subscription.
**created** | **string**<br/><br/>The date and time when the subscription was created (UTC). It uses ISO 8601 format: `YYYY-MM-DDThh:mm:ss[.sss]`.
**has_more** | <GetWebhookSubscriptionsListTable4/>
