**string**<br/><br/>The OS family, for example:

- `WINDOWS`
- `ANDROID`
- `IOS`
- `OS X`
- `LINUX`.<br/><br/>The complete list is available on the [UADetector library page](http://uadetector.sourceforge.net/modules/uadetector-core/apidocs/net/sf/uadetector/OperatingSystemFamily.html).<br/><br/>Value of "operator":

- `EQ` — Equal to.
- `NE` — Not equal to.<br/><br/>Value of "value" if data couldn't be extracted — `UNKNOWN`.
