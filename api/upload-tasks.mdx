# Ways to upload tasks

<Announce/>

## Uploading tasks using the API {#download-tasks-api}

Toloka provides two ways of uploading tasks using the API:

#### Uploading each task individually

Choose this method if:

- You [created a pool](/docs-new/api/create-pool) using ["smart mixing"](https://toloka.ai/en/docs/guide/concepts/distribute-tasks-by-pages#distribute-tasks-by-pages__smart-mixing) (the `mixer_config` parameter).

- Apart from the general tasks, the pool will contain control or training tasks.

With this method, Toloka will split tasks into suites according to the "smart mixing" settings. When uploading tasks this way, use the [POST /tasks](/docs-new/api/create-task) request.

<Cut title="Sample request">

```bash
POST https://toloka.dev/api/v1/tasks
Authorization: OAuth <OAuth token>
Content-Type: application/JSON

// one task
{task data}

// multiple tasks [{task 1}, {task 2},... {task n}]
```

Request body:

#### Uploading tasks grouped by suite

Choose this method if you:

- Create task suites yourself.
- Determine yourself which tasks to include in each suite.

This method won't work if you created a pool using [smart mixing](https://toloka.ai/en/docs/guide/concepts/distribute-tasks-by-pages#distribute-tasks-by-pages__smart-mixing). When uploading tasks this way, use the [POST /task-suites](/docs-new/api/create-task-suite) request.

<Cut title="Sample request">

```bash
POST https://toloka.dev/api/v1/task-suites
Authorization: OAuth <OAuth token>
Content-Type: application/JSON

// one task suite
{task suite parameters}

// multiple task suites [{task suite 1}, {task suite 2},... {task suite N}]
```

Request body:

## Uploading tasks using the Python SDK {#download-tasks-python}

If you're developing Python apps, you can create API requests using [Toloka-Kit](/docs-new/toloka-kit/) methods to achieve a result with less effort and avoid possible errors.

Ways to upload tasks using Toloka-Kit:

Way | Method
----- | -----
Creates a new task. | [create_task](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.create_task)
Creates multiple tasks within a single request. | [create_tasks](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.create_tasks)
Creates multiple tasks asynchronously. | [create_tasks_async](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.create_tasks_async)
Creates a task suite. | [create_tasks_suite](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.create_task_suite)
Creates multiple task suites within a single request. | [create_tasks_suites](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.create_task_suites)
Creates multiple task suites asynchronously. | [create_tasks_suites_async](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.create_task_suites_async)
