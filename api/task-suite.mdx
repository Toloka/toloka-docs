# Overview

A task suite is a set of tasks grouped on a single web page. If the tasks are simple, you can add 10–20 tasks per suite. Don't make task suites too long because it slows down the loading speed for Tolokers.

To create a task suite, upload JSON containing:

- Input data for tasks (such as text or image URLs).
- Correct responses (for control tasks).
- Hints (for training tasks).
- Parameters for assigning tasks.

## Methods {#methods}

Method | Endpoint | Overview
----- | ----- | -----
POST | [/task-suites](/docs-new/api/create-task-suite) | Creates one or multiple task suites.
POST | [/task-suites](/docs-new/api/create-task-suite-batch) | Creates multiple task suites.
GET | [/task-suites](/docs-new/api/get-task-suite-list) | Gets the list of task suites in the pool.
GET | [/task-suites/<task_suite_id>](/docs-new/api/get-task-suite) | Gets a task suite.
PATCH | [/task-suites/<task_suite_id>](/docs-new/api/edit-overlap) | Changes task suite overlap.
PATCH | [/task-suites/<task_suite_id>](/docs-new/api/edit-order) | Changes the priority of a task suite in the pool.
PATCH | [/task-suites/<task_suite_id>/set-overlap-or-min](/docs-new/api/set-min-suite-overlap) | Stops assigning a task suite to Tolokers.
