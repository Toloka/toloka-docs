# Inserting images

This section describes how to insert [one image](/docs-new/template-builder/operations/insert-images), [two images side-by-side](/docs-new/template-builder/operations/insert-images), or an [array of images](/docs-new/template-builder/operations/insert-images).

<TolokaTbSourceAddMedia/>

## One image {#single-picture}

To display an image, you need a direct link to the image and the [view.image](/docs-new/template-builder/reference/view.image) component. You can change the height, width, frame, and other parameters of the image. For more information, see the component description.

To insert an image passed in the input data, use the `data.input` component in the `url` property.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/zDwELDM23ttBiT">View example</Button>

## Two images side-by-side {#side-by-side}

To place two images side-by-side, use the [layout.side-by-side](/docs-new/template-builder/reference/layout.side-by-side) component.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/ZdGClef13ttEz8">View example</Button>

 For a detailed analysis of the example, see [Quick start](../quickstart).

## Array of images {#array}

To get a value from a specific element in an array, use the path to specify its sequence number, starting from zero.

Let's say the input data contains of an array of links to images:
```json
{
  "images": [
    "https://cdn.stocksnap.io/img-thumbs/960w/surfer-wave_3DBOYBPB3X.jpg",
    "https://cdn.stocksnap.io/img-thumbs/960w/fisherman-silhouette_UADULRRHEK.jpg",
    "https://cdn.stocksnap.io/img-thumbs/960w/old-photo_GEQ27OWZVV.jpg"
  ]
}
```
You can reference a specific array element like this:
```json
"url": {
  "type": "data.input",
  "path": "images.<Element number, starting from zero>"
}
```

### Components for displaying a group of images {#additional-components}

To display multiple images, you need a component that you can add your images to. To do this, you can use components that implement a list or table:
- [view.list](/docs-new/template-builder/reference/view.list): Lets you create a list of any elements, including images. It's good for making vertical lists.

    <Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/V44snIfa3ttBb2">View example</Button>

    In horizontal mode (to do this you need to add `"direction": true`), the images are small because they have a restricted height and width. You can increase the image width by setting the minimum width in the `minWidth` property .

    <Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/VBzpJKCp3ttBpt">View example</Button>

- [layout.columns](/docs-new/template-builder/reference/layout.columns): This component is good for displaying a small number of images in the horizontal mode. It lets you align the content vertically, like if you want to position images in the center.

    <Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/zZHgwLrX3ttBrK">View example</Button>

<Note mode="note" title="Note">

This is relevant for creating an interface in Toloka for one task that uses multiple images. Consider whether you need multiple images in the same task. If not, the [view.image](/docs-new/template-builder/reference/view.image) component is enough for your task.

</Note>

### Many images or unknown number of images {#unknown-size}

If you don't know the size of your array with images in advance or you have so many images that the code is too large, use one of the components described above and add the [helper.transform](/docs-new/template-builder/reference/helper.transform) component into the `items` property. This component lets you convert an array of links to an array of [view.image](/docs-new/template-builder/reference/view.image) components to display them in the interface.

## Zooming in on an image {#zoom-image}

To zoom in on an image by pressing a shortcut key, use the [plugin.hotkeys](/docs-new/template-builder/reference/plugin.hotkeys) component. Specify which key triggers [action.open-close](/docs-new/template-builder/reference/action.open-close). The image is specified in the `view` property using the [$ref](/docs-new/template-builder/best-practices/reuse) structure. The action is performed when a Toloker presses the selected key.
