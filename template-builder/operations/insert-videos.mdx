# Inserting videos

This section describes how [to insert a video](#insert-videos) and [make sure the Toloker viewed it](#condition.played).

## Inserting videos {#insert-video}

To add a video file to the task interface, use the [view.video](/docs-new/template-builder/reference/view.video) component. In the `url` property, specify a direct link to the file or component that returns the link. To use the link from the input data, use the [data.input](/docs-new/template-builder/operations/work-with-data) component.

If you want to add several videos to a page, see the examples in [Comparing video clips](/docs-new/template-builder/operations/sbs-video).
