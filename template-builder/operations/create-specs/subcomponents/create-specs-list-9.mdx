In this example, the specification will contain the `verdict` field with the **string** type and the acceptable values "true" and "false".

  ```json
  {
"type": "field.radio-group",
"options": [
  {
    "label": "Yes",
    "value": "true"
  },
  {
    "label": "No",
    "value": "false"
  }
],
"data": {
  "type": "data.output",
  "path": "verdict"
}
  }
  ```
