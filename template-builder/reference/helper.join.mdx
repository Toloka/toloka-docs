# helper.join

The component combines two or more strings into one. You can add a delimiter to separate the strings, such as a space or comma.

- [view.text](/docs/template-builder/reference/view.text): Displays a text.
- [field.select](/docs/template-builder/reference/field.select): Adds a drop-down list.
