# List of plugins

Use the components named `plugin.*` for plugins that enable advanced functionality. For example, use [plugin.hotkeys](/docs-new/template-builder/reference/plugin.hotkeys) to set up shortcuts.

| Component                                                                         | Description                                                                                             |
| --------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| [plugin.field.image-annotation.hotkeys](/docs-new/template-builder/reference/plugin.field.image-annotation.hotkeys) | Used to set hotkeys for the [field.image-annotation](/docs-new/template-builder/reference/field.image-annotation) component.              |
| [plugin.field.text-annotation.hotkeys](/docs-new/template-builder/reference/plugin.field.text-annotation.hotkeys)   | Use this to set keyboard shortcuts for the [field.text-annotation](/docs-new/template-builder/reference/field.text-annotation) component. |
| [plugin.hotkeys](/docs-new/template-builder/reference/plugin.hotkeys)                                               | You can use this to set keyboard shortcuts for actions.                                                 |
| [plugin.toloka](/docs-new/template-builder/reference/plugin.toloka)                                                 | A plugin with extra settings for tasks in Toloka.                                                       |
| [plugin.trigger](/docs-new/template-builder/reference/plugin.trigger)                                               | Use this to configure triggers that trigger a specific action when an event occurs.                     |
