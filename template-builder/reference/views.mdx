# List of views

Use the components named `view.*` for the elements displayed in the interface, such as text, list, audio player, or image. A Toloker can't change these components. Use the [data entry components](/docs-new/template-builder/reference/fields) (fields) where you want Tolokers to enter something.

| Component                                   | Description                                                                                                                     |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| [view.action-button](/docs-new/template-builder/reference/view.action-button) | Button that calls an action.                                                                                                    |
| [view.alert](/docs-new/template-builder/reference/view.alert)                 | The component creates a color block to highlight important information.                                                         |
| [view.audio](/docs-new/template-builder/reference/view.audio)                 | The component plays audio.                                                                                                      |
| [view.collapse](/docs-new/template-builder/reference/view.collapse)           | Expandable block.                                                                                                               |
| [view.device-frame](/docs-new/template-builder/reference/view.device-frame)   | Wraps the content of a component in a frame that is similar to a mobile phone. You can place other components inside the frame. |
| [view.divider](/docs-new/template-builder/reference/view.divider)             | Horizontal delimiter.                                                                                                           |
| [view.group](/docs-new/template-builder/reference/view.group)                 | Groups components visually into framed blocks.                                                                                  |
| [view.iframe](/docs-new/template-builder/reference/view.iframe)               | Displays the web page at the URL in an iframe window.                                                                           |
| [view.image](/docs-new/template-builder/reference/view.image)                 | Displays an image.                                                                                                              |
| [view.labeled-list](/docs-new/template-builder/reference/view.labeled-list)   | Displaying components as a list with labels placed on the left.                                                                 |
| [view.link](/docs-new/template-builder/reference/view.link)                   | Universal way to add a link.                                                                                                    |
| [view.link-group](/docs-new/template-builder/reference/view.link-group)       | Puts links into groups.                                                                                                         |
| [view.list](/docs-new/template-builder/reference/view.list)                   | Block for displaying data in a list.                                                                                            |
| [view.map](/docs-new/template-builder/reference/view.map)                     | Adds a map to your task.                                                                                                        |
| [view.markdown](/docs-new/template-builder/reference/view.markdown)           | Block for displaying text in Markdown.                                                                                          |
| [view.text](/docs-new/template-builder/reference/view.text)                   | Block for displaying text.                                                                                                      |
| [view.video](/docs-new/template-builder/reference/view.video)                 | Player for video playback.                                                                                                      |
