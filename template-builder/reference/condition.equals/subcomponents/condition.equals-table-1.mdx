Original value. If not specified, it uses the value returned by the parent component (the component that contains `condition.equals`).<br/><br/>How to pass a value:

- Specify the value itself, like the number 42 or a string.
- [Get the value from your data](/docs-new/template-builder/operations/work-with-data).
- Refer to another element using `$ref`.
- Use [helpers](/docs-new/template-builder/reference/helpers) and [conditions](/docs-new/template-builder/reference/conditions) to get the value.
