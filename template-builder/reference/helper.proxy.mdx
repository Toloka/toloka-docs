# @yandex-toloka/helper.proxy

You can use this component to download files from Yandex Disk.

To use `@yandex-toloka/helper.proxy`, connect Yandex Disk to your Toloka account and add the proxy by following [the instructions](/docs-new/guide/prepare-data).

Select the component that you want to add, such as [view.image](/docs-new/template-builder/reference/view.image) for an image or [view.audio](/docs-new/template-builder/reference/view.audio) for an audio file. In the `url` property of this component, select the `@yandex-toloka/helper.proxy` type.
