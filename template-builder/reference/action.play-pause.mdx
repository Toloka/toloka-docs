# action.play-pause

This component controls audio or video playback. It stops playback in progress or starts if it is stopped.

For example, this component will allow you to play two videos simultaneously.

<Button themeColor="light" view="default" size="l" type="link" url="https://ya.cc/t/AiIEBqw-3YbMBH">View example</Button>

You can also stop or start playback for some event ([plugin.trigger](/docs-new/template-builder/reference/plugin.trigger)) or by pressing the hotkey ([plugin.hotkeys](/docs-new/template-builder/reference/plugin.hotkeys)).
