# List of layout options

Use the components named `layouts.*` to position elements in the interface, for example, [in columns](/docs-new/template-builder/reference/layout.columns) or [side-by-side](/docs-new/template-builder/reference/layout.side-by-side). If you have more than one element in the interface, these components will help you arrange them the way you want.

| Component                                     | Description                                                                                           |
| --------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| [layout.bars](/docs-new/template-builder/reference/layout.bars)                 | A component that adds top and bottom bars to the content.                                             |
| [layout.columns](/docs-new/template-builder/reference/layout.columns)           | A component for placing content in columns.                                                           |
| [layout.compare](/docs-new/template-builder/reference/layout.compare)           | Use it to arrange interface elements for comparing them. For example, you can compare several photos. |
| [layout.side-by-side](/docs-new/template-builder/reference/layout.side-by-side) | The component displays several data blocks of the same width on a single horizontal panel.            |
| [layout.sidebar](/docs-new/template-builder/reference/layout.sidebar)           | Lets you place the main content block and an adjacent panel with controls on a page.                  |
