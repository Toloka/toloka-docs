# condition.played

Checks the start of playback. Validation will be passed if playback is started. To play media with the `condition.played` check, you can use [view.audio](/docs-new/template-builder/reference/view.audio) and [view.video](/docs-new/template-builder/reference/view.video). The `condition.played` check only works in the player's `validation` property.
