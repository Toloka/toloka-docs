# Get project items

## Description

Returns all the task items of the project with the ID specified in the request.

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/apps-reference/#get-/app-projects/-app_project_id-/items):** more parameter details, request/response examples, and code samples in various programming languages.

</Note>

## Request

method | url
----- | -----
get | <AppProjectsappProjectIditemsGetTable1/>

### Parameters

name | type | required | description
----- | ----- | ----- | -----
app_project_id | path | true | Project ID.
after_id | query | false | ID of the task item used for cursor pagination.
batch_id | query | false | Batch ID.
status | query | false | Item status.
id_gt | query | false | Items with an ID greater than the specified value.
id_gte | query | false | Items with an ID greater than or equal to the specified value.
id_lt | query | false | Items with an ID smaller than the specified value.
id_lte | query | false | Items with an ID smaller than or equal to the specified value.
created_gt | query | false | Items created after the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_gte | query | false | Items created after or on the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_lt | query | false | Items created before the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_lte | query | false | Items created on or before the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
finished_gt | query | false | Items labeled after the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
finished_gte | query | false | Items labeled after or on the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
finished_lt | query | false | Items labeled before the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
finished_lte | query | false | Items labeled before or on the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
sort | query | undefined | <AppProjectsappProjectIditemsGetTable2/>
limit | query | false | The number of items on the page is set to 100 by default. The maximum value is 1000.
