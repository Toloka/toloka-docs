# Get project templates

## Description

Returns the list of all the available project templates.

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/apps-reference/#get-/apps):** more parameter details, request/response examples, and code samples in various programming languages.

</Note>

## Request

method | url
----- | -----
get | <AppsGetTable1/>

### Parameters

name | type | required | description
----- | ----- | ----- | -----
after_id | query | false | The parameter is used for cursor pagination and indicates the first ID that's used when generating a list of objects. It accounts for filters and sorting.
id_gt | query | false | Only with an ID greater than the specified value.
id_gte | query | false | Only with an ID greater than or equal to the specified value.
id_lt | query | false | Only with an ID smaller than the specified value.
id_lte | query | false | Only with an ID smaller than or equal to the specified value.
sort | query | false | <AppsGetTable2/>
limit | query | false | The number of objects on the page is set to 100 by default. The maximum value is 1000.
lang | query | false | ISO 639 language code.
