# Get projects

## Description

Returns the list of all the currently available projects. You can also filter this list using the query parameters.

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/apps-reference/#get-/app-projects):** more parameter details, request/response examples, and code samples in various programming languages.

</Note>

## Request

method | url
----- | -----
get | <AppProjectsGetTable1/>

### Parameters

name | type | required | description
----- | ----- | ----- | -----
app_id | query | false | ID of the bespoke solution that the projects belong to.
parent_app_project_id | query | false | ID of the project used for cloning. This is specified only if you have created a new project by cloning another project. You can clone projects in the web version of Toloka.
status | query | false | Project status.
after_id | query | false | ID of the project used for cursor pagination.
id_gt | query | false | Projects with an ID greater than the specified value.
id_gte | query | false | Projects with an ID greater than or equal to the specified value.
id_lt | query | false | Projects with an ID less than the specified value.
id_lte | query | false | Projects with an ID less than or equal to the specified value.
name_gt | query | false | Projects with a name lexicographically greater than the specified value.
name_gte | query | false | Projects with a name lexicographically greater than or equal to the specified value.
name_lt | query | false | Projects with a name lexicographically smaller than the specified value.
name_lte | query | false | Projects with a name lexicographically smaller than or equal to the specified value.
created_gt | query | false | Projects created after the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_gte | query | false | Projects created after or on the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_lt | query | false | Projects created before the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_lte | query | false | Projects created before or on the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
sort | query | false | <AppProjectsGetTable2/>
limit | query | false | The number of objects on the page is set to 100 by default. The maximum value is 5000.
scope | query | false | Projects created by a specified range of requesters: MY - Only by me; COMPANY - By anyone from the company; REQUESTER_LIST - By requesters with the specified IDs.
requester_ids | query | false | List of requester IDs separated by a comma, for scope = REQUESTER_LIST.
