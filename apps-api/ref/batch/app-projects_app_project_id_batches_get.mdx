# Get project batches

## Description

Returns the complete list of the batches associated with the project with the ID specified in the request.

<Note mode="tip" title="Tip">

**Try our [new API reference](https://toloka.ai/docs/api/apps-reference/#get-/app-projects/-app_project_id-/batches):** more parameter details, request/response examples, and code samples in various programming languages.

</Note>

## Request

method | url
----- | -----
get | <AppProjectsappProjectIdbatchesGetTable1/>

### Parameters

name | type | required | description
----- | ----- | ----- | -----
app_project_id | path | true | Project ID.
after_id | query | false | ID of the batch used for cursor pagination.
status | query | false | Batch status.
id_gt | query | false | Batches with an ID greater than the specified value.
id_gte | query | false | Batches with an ID greater than or equal to the specified value.
id_lt | query | false | Batches with an ID smaller than the specified value.
id_lte | query | false | Batches with an ID smaller than or equal to the specified value.
name_gt | query | false | Batches with a name lexicographically greater than the specified value.
name_gte | query | false | Batches with a name lexicographically greater than or equal to the specified value.
name_lt | query | false | Batches with a name lexicographically smaller than the specified value.
name_lte | query | false | Batches with a name lexicographically smaller than or equal to the specified value.
created_gt | query | false | Batches created after the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_gte | query | false | Batches created after or on the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_lt | query | false | Batches created before the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
created_lte | query | false | Batches created on or before the specified date. The date and time is specified in UTC in ISO 8601 format: YYYY-MM-DDThh:mm:ss[.sss].
sort | query | false | <AppProjectsappProjectIdbatchesGetTable2/>
limit | query | false | The number of batches on the page is set to 100 by default. The maximum value is 1000.
