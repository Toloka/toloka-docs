Merging multiple arrays into a single array.

For example, let's say you have multiple arrays:
([1, 2, 3], [4, 5, 6], [7, 8, 9])
Their elements can be combined into a single array to show simultaneously:
[1, 2, 3, 4, 5, 6, 7, 8, 9]
