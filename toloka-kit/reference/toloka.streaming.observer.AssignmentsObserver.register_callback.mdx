# register_callback
`toloka.streaming.observer.AssignmentsObserver.register_callback` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.1.3/src/streaming/observer.py#L356)

```python
register_callback(
    self,
    callback: Union[Callable[[List[AssignmentEvent]], None], Callable[[List[AssignmentEvent]], Awaitable[None]]],
    event_type: AssignmentEvent.Type
)
```

Register given callable for given event type.

Callback will be called multiple times if it has been registered for multiple event types.

## Parameters Description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`callback`|**Union\[Callable\[\[List\[[AssignmentEvent](/docs-new/toloka-kit/reference/toloka.streaming.event.AssignmentEvent)\]\], None\], Callable\[\[List\[[AssignmentEvent](/docs-new/toloka-kit/reference/toloka.streaming.event.AssignmentEvent)\]\], Awaitable\[None\]\]\]**|<p>Sync or async callable that pass List[AssignmentEvent] of desired event type.</p>
`event_type`|**[AssignmentEvent.Type](/docs-new/toloka-kit/reference/toloka.streaming.event.AssignmentEvent.Type)**|<p>Selected event type.</p>

* **Returns:**

  The same callable passed as callback.
