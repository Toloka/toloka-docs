# find_assignments
`toloka.client.TolokaClient.find_assignments` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.1.3/src/client/__init__.py#L654)

Finds assignments that match certain criteria.

The number of returned assignments is limited. To find remaining assignments call `find_assignments` with updated search criteria.

To iterate over all matching assignments you may use the [get_assignments](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.get_assignments) method.
