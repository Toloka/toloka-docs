# SkillId
`toloka.client.conditions.SkillId` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.1.3/src/client/conditions.py#L297)

SkillId(

The ID of a changed skill which caused access blocking.

`SkillId` provides details if the [PoolAccessRevokedReason](/docs-new/toloka-kit/reference/toloka.client.conditions.PoolAccessRevokedReason) condition equals `SKILL_CHANGE`.

`SkillId` is used with collectors:
- [UsersAssessment](/docs-new/toloka-kit/reference/toloka.client.collectors.UsersAssessment)
