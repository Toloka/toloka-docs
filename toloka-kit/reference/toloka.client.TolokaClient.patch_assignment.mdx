# patch_assignment
`toloka.client.TolokaClient.patch_assignment` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.1.3/src/client/__init__.py#L731)

Changes an assignment status and associated public comment.

See also [reject_assignment](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.reject_assignment) and [accept_assignment](/docs-new/toloka-kit/reference/toloka.client.TolokaClient.accept_assignment).

## Parameters Description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`assignment_id`|**str**|<p>The ID of the assignment.</p>
`public_comment`|**Optional\[str\]**|<p>The public comment.</p>
`status`|**Optional\[[Assignment.Status](/docs-new/toloka-kit/reference/toloka.client.assignment.Assignment.Status)\]**|<p>The new status of an assignment:</p> <ul> <li>`ACCEPTED` — Accepted by the requester.</li> <li>`REJECTED` — Rejected by the requester.</li> </ul>

* **Returns:**

  Assignment object with updated fields.

* **Return type:**

  [Assignment](/docs-new/toloka-kit/reference/toloka.client.assignment.Assignment)

**Examples:**

```python
toloka_client.patch_assignment(assignment_id='1', public_comment='Accepted. Good job.', status='ACCEPTED')
```
