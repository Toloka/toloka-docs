# open_pool
`toloka.async_client.client.AsyncTolokaClient.open_pool` | [Source code](https://github.com/Toloka/toloka-kit/blob/v1.1.3/src/async_client/client.py#L0)

```python
async open_pool(self, pool_id: str)
```

Opens a pool.

After opening the pool, tasks can be assigned to Tolokers.

## Parameters Description

| Parameters | Type | Description |
| :----------| :----| :-----------|
`pool_id`|**str**|<p>The ID of the pool.</p>

* **Returns:**

  The pool with updated status.

* **Return type:**

  [Pool](/docs-new/toloka-kit/reference/toloka.client.pool.Pool)

**Examples:**

Opening a pool.

```python
toloka_client.open_pool(pool_id='1')
```
