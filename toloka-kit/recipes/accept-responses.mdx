# Accept responses from Tolokers

_Accept Toloker responses with the IDs specified in the request._

## Steps to follow

<Import/>

<Instantiate/>

### 3. Accept Toloker response {#step-three}

[Find out](/docs/toloka-kit/recipes/get-responses) the ID of the Toloker response that your want to accept and use the [accept_assignment()](/docs/toloka-kit/reference/toloka.client.TolokaClient.accept_assignment) method to accept the response and leave a comment.

<CodeCopy
  isSmall
  theme="light"
  code={
    <HighlightChildren language="python">
      accepted_assignment = toloka_client.<Term term="accept_assignment"><AcceptAsignment/></Term>('0001d38f5b--61c8be211c3a7842a596ac0a', 'OK');
    </HighlightChildren>
  }
  textToCopy="accepted_assignment = toloka_client.accept_assignment('0001d38f5b--61c8be211c3a7842a596ac0a', 'OK')"
/>

The Toloker response will be marked as accepted with the `OK` comment added to it.

### 4. Print Toloker response status and public comment {#step-four}

The `accept_assignment()` request will return the [Assignment](/docs/toloka-kit/reference/toloka.client.assignment.Assignment) class object. You can use its attributes to print the information you need.

```python
print(accepted_assignment.status, accepted_assignment.public_comment)
```

You should get an output with the updated Toloker response status and public comment which looks like this.

```bash
Status.ACCEPTED OK
```

## Complete code: Accept responses from Tolokers {#complete-code}

```python
import toloka.client as toloka

toloka_client = toloka.TolokaClient('AQC2AGAJgyNSA8CtpdO9MWy_QEB6s6kDjHUoElE', 'PRODUCTION')

accepted_assignment = toloka_client.accept_assignment('0001d38f5b--61c8be211c3a7842a596ac0a', 'OK')
print(accepted_assignment.status, accepted_assignment.public_comment)
```
