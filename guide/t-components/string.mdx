# String input field

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a string input field in [Template Builder](/docs-new/template-builder/reference/field.text).

</Note>

To add a string input field to the task, use the `{{field type="input" name="<output field> name"}}` component. Example:

```plaintext
{{field type="input" name="result" placeholder="Enter a word" validation-show="right-center"}}
```

Add a field of `string` type in the [output data description](/docs-new/guide/incoming). Example:

```json
{
  "result": {
    "type": "string",
    "required": true
     }
}
```
