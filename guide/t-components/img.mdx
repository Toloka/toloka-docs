# Image

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try adding an image in [Template Builder](/docs-new/template-builder/operations/insert-images).

</Note>

To add an image to the task, use the `{{img src=<image URL>}}` component. Example:

```plaintext
{{img src=url width="100px" height="75px"}}
```

You [can store your files in Yandex Cloud](/docs-new/guide/use-object-storage).

## Parameters

**Parameter**|**Description**|**Required**|**Default value**
----- | ----- | ----- | -----
`src`| <ImgTable1/>| yes | no
`width`| <ImgTable2/>| no | `"300px"`
`height`| <ImgTable3/>| no | `"300px"`
`real-size`| <ImgTable4/>| no | `false`
`class`| CSS class for the image. | no | `".img"`
