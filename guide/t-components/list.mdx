# Drop-down list

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a drop-down list in [Template Builder](/docs-new/template-builder/reference/field.select).

</Note>

To add a dropdown list to the task, use `{{#field type="select" name="<output field name>"}}... {{/field}}`. The entries in the dropdown list are created using the nested expression ``{{select_item}}``. Example:

```plaintext
{{#field type="select" name="colour" placeholder="Choose a color" width="200px"}}
{{select_item value="red" text="Red"}}
{{select_item value="green" text="Green"}} {{/field}}
```

Add a `string` field to the [output data description](/docs-new/glossary#input-output-data).

```json
{
  "colour": {
    "type": "string",
    "required": true
  }
}
```
