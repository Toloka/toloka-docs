A condition for performing the action in the **then** field:

- **Toloker banned** — The Toloker's access to tasks is blocked by a [quality control rule](/docs-new/glossary#quality-control-rule) (such as [control tasks](/docs-new/glossary#control-task), [majority vote](/docs-new/glossary#majority-vote), [fast responses](/docs-new/glossary#fast-responses), or [skipping tasks](/docs-new/glossary#skipping-tasks)).

- **Toloker filtered out** — The Toloker no longer meets one or more [filters](/docs-new/guide/filters).
