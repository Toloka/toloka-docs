# Remove a skill

You can remove a skill only manually. It can't be done using quality control rules.

1. Go to the [Tolokers](https://platform.toloka.ai/requester/workers) page.

1. Select the Tolokers to remove the skill from.

    <SelectTolokers/>

1. Click **-Skill**.

1. Select a skill.

1. Click **Delete**.
