# Archiving a pool

A [pool](/docs-new/glossary#pool) with the “Archived” status can't be started or edited and isn't available for the [task review](/docs-new/guide/accept).

By default, archived pools are not visible in the list of [project](/docs-new/glossary#project) pools. To view them:

1. Open the project page.

1. In the **Pools** tab, select **Archived**.

The pool is automatically archived if none of the following actions is performed in the pool for a month:

- Editing.
- Uploading tasks.
- Changing status.
- Completing a task.
- Cloning via the Toloka API.

To archive a pool, click **<img src="https://tlk-infra-front.azureedge.net/portal-static/uploaded/images/1675425165980/1675425165980_original_svg.svg" alt="Drop-down button" /> → Archive** at the top of the pool page, or <img src="https://tlk-infra-front.azureedge.net/portal-static/uploaded/images/1675425183285/1675425183285_original_svg.svg" alt="" /> in the list of pools on the project page.

<Note mode="note" title="Note">

To [archive the training pool](/docs-new/guide/train), first archive all the main pools to which it is linked.

</Note>
