1. Select a Toloker on the [Tolokers](https://platform.toloka.ai/requester/workers) page.

To block access to tasks for multiple Tolokers:

1. Select Tolokers by using the [filters](/docs-new/glossary#filters) on the [Tolokers](https://platform.toloka.ai/requester/workers) page or upload a TSV file:

    ```plaintext
    <annotator id 1>
    <annotator id 2>
    ...
    <annotator id n>
    ```

1. Click **Ban**, then fill in the fields (see the table above).

You can view information about access to tasks on the Toloker's page (on the [Tolokers](https://platform.toloka.ai/requester/workers) page, go to the **Bans** tab).
