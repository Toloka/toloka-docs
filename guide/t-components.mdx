# Special components and helpers

<Note mode="note" title="Note">

The task interface configuration guide describes the features of the HTML/JS/CSS editor. You can also try creating a task interface in [Template Builder](/docs-new/template-builder/reference/).

</Note>

To create a [task interface](/docs-new/glossary#task-interface), go to the [HTML](/docs-new/guide/spec#html) section on the [project](/docs-new/glossary#project) page. Here's what you can use:

- [Handlebars expressions and helpers](/docs-new/guide/t-components/handlebars) (see the [documentation on the Handlebars website](http://handlebarsjs.com/)).

- Additional helpers:

    - [Concatenation](/docs-new/guide/t-components/helpers#concat)
    - [Object operations](/docs-new/guide/t-components/helpers#object)
    - [Logical operations and comparisons](/docs-new/guide/t-components/helpers#equal)
    - [Recording output data field values in JSON](/docs-new/guide/t-components/helpers#js_fields)

- Components for solving common problems:

    - [String input field](/docs-new/guide/t-components/string)
    - [Text input field](/docs-new/guide/t-components/text)
    - [String input with suggest list](/docs-new/guide/t-components/suggest)
    - [Switch](/docs-new/guide/t-components/radiobuttons)
    - [Checkbox](/docs-new/guide/t-components/checkboxes)
    - [Dropdown list](/docs-new/guide/t-components/list)
    - [Button for file upload](/docs-new/guide/t-components/upload-file)
    - [Button for image upload](/docs-new/guide/t-components/upload-picture)
    - [Button for recording and uploading an audio file](/docs-new/guide/t-components/upload-audio)
    - [Button with click validation](/docs-new/guide/t-components/button-link)
    - [Image with area selection](/docs-new/guide/t-components/image-annotation)
    - [Text segmentation editor](/docs-new/guide/t-components/segmentation-editor)
    - [Image](/docs-new/guide/t-components/img)
    - [Button](/docs-new/guide/t-components/button)
    - [Iframe](/docs-new/guide/t-components/iframe)
    - [Text with HTML formatting](/docs-new/guide/t-components/html)

Libraries for using Handlebars and additional components are connected by default when you create a project.
